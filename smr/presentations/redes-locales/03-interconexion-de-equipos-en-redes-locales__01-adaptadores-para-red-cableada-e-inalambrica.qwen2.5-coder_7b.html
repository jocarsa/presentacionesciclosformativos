Objetivo: Comprender los adaptadores para red cableada e inalámbrica.

Adaptadores para red cableada

  • Conectan dispositivos a redes Ethernet
  • Tipo de conexión: Coaxial, Twisted Pair (Cat 5/6)
  • Ejemplos: NICs (Network Interface Cards)
  • Funciones principales: Conversión de señales y gestión de datos

Adaptadores para red inalámbrica

  • Permiten conexión a redes Wi-Fi
  • Tecnologías: 802.11a, b, g, n, ac
  • Ejemplos: Modem-router inalámbrico
  • Funciones principales: Transmisión de señales inalámbricas y gestión de la red

Tipo de adaptadores para red cableada

  • NICs (Network Interface Cards)
  • PCIe
  • USB
  • Ethernet
  • Coaxial

Tipo de adaptadores para red inalámbrica

  • Modem-router inalámbrico
  • Adaptador USB Wi-Fi
  • Antenas inalámbricas
  • Receptores inalámbricos

Ejemplo de NIC (Network Interface Card)

NICs son adaptadores para red cableada que permiten la conexión a redes Ethernet.

Ejemplo de Modem-router inalámbrico

Este dispositivo combina funciones de modem y router, permitiendo conexión a Internet e inalámbrica.

Buenas prácticas al conectar adaptadores para red

  • Verificar la compatibilidad del adaptador con el sistema operativo
  • Instalar los controladores adecuados
  • Configurar correctamente las direcciones IP
  • Mantener el software actualizado

Errores comunes al conectar adaptadores para red

  • No detecta la conexión
  • Dirección IP incorrecta
  • Problemas de seguridad
  • Interferencias inalámbricas

Comparativa: Adaptadores para red cableada vs. inalámbrica
