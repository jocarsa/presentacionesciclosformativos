Instalación/configuración de los equipos de red

Objetivo: Comprender y aplicar procedimientos de instalación y protocolos en redes locales.

Introducción a la Instalación de Equipos de Red

  • Definición de equipos de red
  • Tipo de dispositivos comunes (switches, routers, firewalls)
  • Importancia de la configuración correcta

Proceso de Instalación Básico

  1. Selección del equipo adecuado
  2. Configuración física (conexiones, puertos)
  3. Instalación del software necesario

Ejemplo de Instalación: Configuración de un Switch

  • Conectar el switch a la fuente de alimentación
  • Configurar las puertos según necesidad
  • Instalar software de gestión (como SNMP)

Protocolos Básicos en Redes Locales

  • IP (Internet Protocol)
  • Subnetting
  • DNS (Domain Name System)

Ejemplo de Configuración IP en Windows

  1. Ir a "Configuración de red"
  2. Seleccionar la interfaz de red
  3. Configurar dirección IP, máscara y puerta de enlace

Caso Práctico: Configuración de un Router

  • Conectar el router a la fuente de alimentación
  • Configurar la dirección IP del router
  • Establecer puertas de enlace y DNS

Buenas Prácticas en Instalación de Redes

  • Seguridad física (ubiertas,ajas)
  • Etiquetado claro de cables y puertos
  • Mantenimiento regular del equipo

Errores Comunes en la Instalación de Redes

  • No conectar correctamente los cables
  • Olvidar configurar la dirección IP
  • No establecer puertas de enlace y DNS

Comparativa entre Switches y Routers

Switch Router

Checklist de Instalación de Redes Locales

  1. Verificar la conectividad física
  2. Configurar las direcciones IP
  3. Establecer puertas de enlace y DNS
  4. Realizar pruebas de red (ping, traceroute)
