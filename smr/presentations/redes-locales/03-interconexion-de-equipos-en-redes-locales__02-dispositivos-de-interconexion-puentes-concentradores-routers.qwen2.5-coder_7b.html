Objetivo: Comprender los dispositivos de interconexión utilizados en redes locales.

Definición de red local

Una red local es un sistema de comunicación que permite la conexión de dos o más computadoras y otros dispositivos en una misma ubicación física.

Tipos de redes locales

  • Red Local Inalámbrica (WLAN)
  • Red Local Conectada porables (LAN)
  • Red Metropolitana (MAN)
  • Red Nacional (NAN)

Puente Ethernet

Un puente Ethernet es un dispositivo de red que conecta dos o más segmentos de una misma red.

Ejemplo de uso: Puente Ethernet

Permite la comunicación entre computadoras en diferentes partes del mismo edificio.

Puerto de puente Ethernet

  • Porta 10Base-T
  • Porta 100Base-TX
  • Porta 1000Base-T (Gigabit)

Puente Ethernet vs. Concentrador

Puente Ethernet Concentrador
Velocidad 10/100 Mbps 10/100 Mbps
Estructura de red Topología de anillo o estrella Topología de estrella

Concentrador Ethernet

Un concentrador Ethernet es un dispositivo que permite la conexión de varios dispositivos a una misma red.

Ejemplo de uso: Concentrador Ethernet

Permite la comunicación entre computadoras en el mismo área local.

Funciones del concentrador Ethernet

  • Repetir señales eléctricas
  • Distribuir datos a todos los dispositivos conectados
  • Controlar la velocidad de transmisión

Routers Ethernet

Un router Ethernet es un dispositivo que permite la comunicación entre diferentes redes.

Ejemplo de uso: Router Ethernet

Permite la conexión a Internet y compartir recursos entre diferentes redes.

Funciones del router Ethernet

  • Ruteo de paquetes IP
  • Conexión a Internet
  • Compartir recursos entre redes

Comparación puente, concentrador y router

Puente Concentrador Router
Tipo de conexión Ethernet Ethernet Ethernet, Wi-Fi
Función principal Interconectar segmentos de red Distribuir datos a todos los dispositivos Rutear paquetes IP entre redes

Buenas prácticas en el uso de puentes, concentradores y routers

  • Configurar correctamente la dirección IP
  • Usar cables Ethernet de buena calidad
  • Mantener los dispositivos actualizados
  • Evitar sobrecargar el router con demasiados dispositivos

Errores comunes en la interconexión de redes locales

  • No conectar los cables correctamente
  • No configurar las direcciones IP adecuadamente
  • No mantener los dispositivos actualizados
  • Sobrecargar el router con demasiados dispositivos

Mini-quiz: Dispositivos de interconexión

¿Cuál es la principal función del puente Ethernet?

  1. Distribuir datos a todos los dispositivos conectados
  2. Rutear paquetes IP entre redes
  3. Interconectar segmentos de red
  4. Conectar a Internet y compartir recursos entre diferentes redes

Checklist: Configuración de puente Ethernet

  • Conectar los cables Ethernet correctamente
  • Configurar la dirección IP del puente
  • Verificar que todos los dispositivos estén conectados
  • Mantener el puente actualizado

Resumen

  • Puente Ethernet: Interconectar segmentos de red
  • Concentrador Ethernet: Distribuir datos a todos los dispositivos
  • Router Ethernet: Rutear paquetes IP entre redes

Normativa general

La normativa vigente en España para las redes locales incluye la regulación de la velocidad de transmisión, la seguridad y la protección contra interferencias electromagnéticas.
