Conectores de red

  • Ethernet (RJ45)
  • Coaxial (F)
  • Serial (DB9)
  • FireWire (IEEE 1394)
  • USB

Ejemplo de Ethernet

Se utiliza para conectar dispositivos Ethernet a una red.

Caso práctico: Conexión de impresoras

La mayoría de las impresoras modernas utilizan conectores USB o Ethernet.

Proceso de instalación de un conector Ethernet

  1. Desconectar el cable Ethernet del switch o router.
  2. Insertar el extremo femenino en la toma de red.
  3. Conectar el otro extremo al puerto RJ45 del dispositivo.
  4. Repetir para todos los dispositivos necesarios.

Buenas prácticas en la instalación de conectores Ethernet

  • Asegúrate de que el cable esté bien aterrado.
  • Evita sobrecargas eléctricas.
  • Mantén los cables ordenados y libres de obstáculos.

Errores comunes en la instalación de conectores Ethernet

  • No aterrar adecuadamente el cable.
  • Sobrecargar los enchufes.
  • Cables entrelazados o cruzados.

Comparativa de conectores Ethernet y coaxial

Ethernet (RJ45) Coaxial (F)
Datos digitales Datos analógicos
Velocidad más alta Distancia mayor
