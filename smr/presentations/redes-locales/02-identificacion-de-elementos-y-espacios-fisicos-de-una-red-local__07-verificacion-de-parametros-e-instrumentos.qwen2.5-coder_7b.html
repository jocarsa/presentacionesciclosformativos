Identificación de elementos y espacios físicos de una red local

Objetivo: Comprender los componentes físicos y cómo verificarlos en una red local.

Definición de Red Local

Una red local es un conjunto de dispositivos interconectados que comparten recursos y datos dentro de un área geográfica limitada.

Componentes Físicos de una Red Local

  • Computadoras (ordenadores, servidores)
  • Switches y Routers
  • Impresoras y dispositivos de almacenamiento
  • Cableado Ethernet o Wi-Fi
  • Puerto de red en los ordenadores

Caso Práctico: Identificación de Componentes

En una red local doméstica, identifica los componentes físicos presentes en tu hogar.

Proceso de Verificación

  1. Listar todos los dispositivos conectados a la red.
  2. Chequear el tipo y estado de cada dispositivo.
  3. Verificar la conexión física (ables Ethernet o Wi-Fi).
  4. Comprobar la configuración IP de los ordenadores.

Buenas Prácticas en Verificación

  • Usar un cable Ethernet para conexiones estáticas.
  • Configurar las IPs de los ordenadores automáticamente si es posible.
  • Mantener el cableado organizado y libre de interferencias.

Errores Comunes en la Verificación

  • No verificar la conexión física.
  • No configurar las IPs correctamente.
  • Mantener el cableado desorganizado y sucio.
