TCP/IP: Estructura y Clases IP

El protocolo TCP/IP es la base de las redes modernas. Se compone de dos capas principales: Transporte (TCP) e Internet (IP).

Estructura de TCP/IP

  • Aplicación
  • Transporte (TCP)
  • Internet (IP)
  • Datos Link
  • Física

Clases IP

TCP/IP utiliza clases de direcciones IP para categorizar y gestionar los dispositivos en la red.

Clase A

  • 1.0.0.0 - 126.255.255.255
  • Usada para redes grandes (127 direcciones de red)
  • Formato: 1xxx.xxxx.xxxx.xxxx

Clase B

  • 128.0.0.0 - 191.255.255.255
  • Usada para redes medias (65,534 direcciones de red)
  • Formato: 1xxx.xxx.xxxx.xxxx

Clase

  • 192.0.0.0 - 223.255.255.255
  • Usada para redes pequeñas (2,097,150 direcciones de red)
  • Formato: 1xxx.xxx.xxx.xxxx

Clase D

  • 224.0.0.0 - 239.255.255.255
  • Usada para multicast (no asignadas a dispositivos individuales)

Clase E

  • 240.0.0.0 - 255.255.255.255
  • Reservadas para futuras extensiones de la arquitectura IP

Ejemplo: Asignación de direcciones IP

Dado un dispositivo en una red B, asigna su dirección IP.

Proceso de asignación

  1. Identificar la clase del rango de direcciones disponibles.
  2. Asignar la primera dirección disponible en el rango para el dispositivo.
  3. Configurar la máscara de subred correspondiente.

Caso: Problema de asignación IP

Un dispositivo no puede conectarse a Internet. ¿Qué podría ser el problema?

Potenciales problemas

  • Dirección IP incorrecta
  • Máscara de subred errónea
  • No asignación de puerta de enlace predeterminada
  • Problemas con el servidor DNS

Buenas prácticas

  • Usar direcciones IP estáticas para dispositivos críticos.
  • Asignar rangos de direcciones IP reservados para redes internas.
  • Mantener actualizadas las máscaras de subred y puertas de enlace.

Errores comunes

  • Confusión entre clases A, B y.
  • No asignación correcta de la máscara de subred.
  • Uso incorrecto de direcciones de clase D y E.

Comparativa: Direcciones IP vs. MAC

Dirección IP Dirección MAC
4 bytes (32 bits) 6 bytes (48 bits)
Usada en la capa de Internet Usada en la capa de datos link

Checklist: Configuración de IP

  • Identificar la clase de dirección IP.
  • Asignar la dirección IP correcta.
  • Configurar la máscara de subred.
  • Establecer la puerta de enlace predeterminada.
  • Verificar la configuración con ping y tracert.

Resumen

  • TCP/IP es la base de las redes modernas.
  • Existen cuatro clases de direcciones IP (A, B,, D y E).
  • Cada clase tiene un rango específico y una máscara de subred predeterminada.

Normativa general

La asignación y gestión de direcciones IP deben seguir las normativas estableidas por la IETF (Internet Engineering Task Force).
