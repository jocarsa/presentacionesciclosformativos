Medios de Transmisión

  • Par trenzado: Medio físico utilizado para la transmisión eléctrica.
  • Fibra óptica: Medio utilizado para la transmisión de luz.

Par Trenzado

Características principales:

  • Se compone de dos hilos de cobre trenzados juntos.
  • Capacidad para transmitir datos a velocidades altas.
  • Susceptible a interferencias electromagnéticas.
  • Menor costo que la fibra óptica.

Fibra Óptica

Características principales:

  • Se compone de un núcleo de vidrio envuelto en una capa de plástico.
  • Capacidad para transmitir datos a velocidades extremadamente altas.
  • Inmunidad a interferencias electromagnéticas.
  • Más costoso que el par trenzado.

Ejemplo de Par Trenzado

Un ejemplo común es el cable Ethernet.

Ejemplo de Fibra Óptica

Se utiliza en redes de alta velocidad, como las de Internet.

Caso Práctico: Uso del Par Trenzado

En una pequeña oficina, el par trenzado se usa para conectar los computadoras a un switch.

Caso Práctico: Uso de la Fibra Óptica

En una instalación industrial, la fibra óptica es utilizada para conectar equipos en diferentes edificios.

Proceso de Instalación del Par Trenzado

  1. Medir y marcar el camino.
  2. Cavar un conducto si es necesario.
  3. Instalar el par trenzado.
  4. Conectar los terminales.

Proceso de Instalación de la Fibra Óptica

  1. Medir y marcar el camino.
  2. Cavar un conducto si es necesario.
  3. Instalar el cable de fibra óptica.
  4. Conectar los terminales.

Buenas Prácticas para el Par Trenzado

  • Usar cable de alta calidad.
  • Mantener la instalación limpia y ordenada.
  • Evitar exponer al cable a daños físicos.

Buenas Prácticas para la Fibra Óptica

  • Usar cable de alta calidad.
  • Mantener la instalación limpia y ordenada.
  • Evitar exponer al cable a daños físicos.

Errores Comunes con el Par Trenzado

  • No usar cable de alta calidad.
  • No mantener la instalación limpia y ordenada.
  • Exponer al cable a daños físicos.

Errores Comunes con la Fibra Óptica

  • No usar cable de alta calidad.
  • No mantener la instalación limpia y ordenada.
  • Exponer al cable a daños físicos.
