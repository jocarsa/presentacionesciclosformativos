Definición del Software Malicioso

Software malicioso es cualquier programa diseñado con el propósito de causar daño, interferir o obtener información sin autorización.

Clasificación del Software Malicioso

  • Virus informáticos
  • Worms
  • Trojanos
  • Ransomware
  • Sware y adware
  • Phishing

Ejemplo de Virus Informático

Ejemplo famoso: Conficker. Se propagaba a través de vulnerabilidades en sistemas Windows.

Worms

Programas autónomos que se replican y se propagan sin necesidad de interacción con el usuario.

Trojanos

Software que oculta su naturaleza maliciosa y se introduce en un sistema bajo la pretensión de ser útil o divertido.

Ransomware

Malware que bloquea el acceso a los archivos del usuario y solicita una rescate por su desbloqueo.

Sware y Adware

  • Sware: Programas que recopilan información sobre las actividades de un usuario sin su conocimiento.
  • Adware: Software que muestra publicidad en la pantalla del usuario.

Phishing

Técnica de engaño mediante el envío de correos electrónicos o mensajes de texto falsificados para obtener información confidencial.

Herramientas de Protección contra Software Malicioso

  • Antivirus
  • Servicios antivirus en la nube
  • Filtros web
  • Firewalls
  • Software de seguridad contra ransomware

Ejemplo de Antivirus

Ejemplos populares: Avast, Bitdefender, Norton.

Servicios Antivirus en la Nube

Ofrecen protección contra amenazas sin necesidad de instalar software localmente.

Filtros Web

Software que controla el acceso a sitios web para prevenir la navegación a páginas maliciosas.

Firewalls

Sistemas de seguridad que monitorean y controlan el tráfico entrante y saliente en una red.

Software contra Ransomware

Ejemplos: Emsisoft Emergency Kit, Malwarebytes Anti-Ransomware.

Buenas Prácticas de Seguridad Informática

  • Mantén el software actualizado
  • No descargues archivos desconocidos
  • Usa contraseñas fuertes y únicas
  • Escanee regularmente tu sistema
  • Realiza copias de seguridad periódicas

Errores Comunes en la Protección contra Malware

  • No usar software gratuito solo porque es gratuito
  • No abrir correos electrónicos de remitentes desconocidos
  • No descargar aplicaciones desde fuentes no confiables
  • No ignorar advertencias del antivirus

Checklist: Protección contra Software Malicioso

  1. Mantén el software actualizado
  2. No descargues archivos desconocidos
  3. Usa contraseñas fuertes y únicas
  4. Escanee regularmente tu sistema
  5. Realiza copias de seguridad periódicas

Checklist: Desinfección

  1. Identifica el malware
  2. Desinstala el software malicioso
  3. Escanee nuevamente con un antivirus confiable
  4. Realiza una limpieza manual de archivos sospechosos

Actividad: Identificación de Malware

Identifica el tipo de malware en las siguientes situaciones:

  • Un correo electrónico que solicita información bancaria.
  • Un archivo descargado desde una fuente desconocida.
  • Una aplicación que se instala automáticamente sin tu consentimiento.

Resumen

El software malicioso es un grave problema de seguridad informática. Se clasifica en varios tipos y requiere herramientas adecuadas para su protección y desinfección.

Normativa General

La normativa vigente establece requisitos mínimos de seguridad informática, incluyendo la protección contra malware. Es responsabilidad de las organizaciones implementar medidas efectivas para proteger sus sistemas.
