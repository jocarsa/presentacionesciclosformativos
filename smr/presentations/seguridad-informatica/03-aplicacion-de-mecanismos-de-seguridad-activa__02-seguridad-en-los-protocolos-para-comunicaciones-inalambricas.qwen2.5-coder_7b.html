Objetivo: Comprender los protocolos para comunicaciones inalámbricas y su seguridad.

Protocolos para comunicaciones inalámbricas

  • Wi-Fi (IEEE 802.11)
  • ZigBee (IEEE 802.15.4)
  • Bluetooth (IEEE 802.15.1)
  • NFC (Near Field Communication)

Características de los protocolos inalámbricos

  • Velocidad de transmisión
  • Rango de cobertura
  • Potencia consumida
  • Canales disponibles
  • Seguridad integrada

Ejemplo: Wi-Fi

Wi-Fi utiliza la banda de radiofrecuencia para transmitir datos a velocidades altas.

Ejemplo: ZigBee

ZigBee es ideal para redes domésticas y de control inteligente con baja velocidad pero alta eficiencia energética.

Ejemplo: Bluetooth

Bluetooth permite la conexión inalámbrica a corta distancia para dispositivos móviles.

Ejemplo: NFC

NFC es ideal para transacciones rápidas y interambio de datos entre dispositivos cercanos.

Seguridad en Wi-Fi

  • WEP (Wired Equivalent Privacy)
  • WPA (Wi-Fi Protected Access)
  • WPA2 (Wi-Fi Protected Access II)
  • WPA3 (Wi-Fi Protected Access III)

Seguridad en ZigBee

ZigBee utiliza AES-128 para cifrado de datos.

Seguridad en Bluetooth

Bluetooth utiliza AES-128 para cifrado de datos y autenticación de dispositivos.

Seguridad en NFC

NFC utiliza AES-128 para cifrado de datos.

Caso: Ataque a Wi-Fi WEP

El ataque WEP Cracking puede descifrar el tráfico Wi-Fi sin autorización.

Buenas prácticas en seguridad inalámbrica

  • Cambiar la contraseña de red regularmente
  • Desactivar el modo de "reconocimiento" del router
  • Usar protocolos seguros (WPA3)
  • Limitar los dispositivos autorizados en la red

Errores comunes en seguridad inalámbrica

  • No cambiar la contraseña predeterminada del router
  • Habilitar el modo de "reconocimiento" del router
  • Usar protocolos obsoletos (WEP, WPA)
  • No limitar los dispositivos autorizados en la red

Comparativa de seguridad entre protocolos

Protocolo Cifrado Autenticación Rango
Wi-Fi WEP AES-40/128 Clave compartida 100 metros
Wi-Fi WPA AES-128 Pre-Shared Key / EAP 100 metros
Wi-Fi WPA2 AES-128/256 Pre-Shared Key / 802.1X 100 metros
Wi-Fi WPA3 AES-192/256 Pre-Shared Key / 802.1X 100 metros

Checklist de seguridad inalámbrica
