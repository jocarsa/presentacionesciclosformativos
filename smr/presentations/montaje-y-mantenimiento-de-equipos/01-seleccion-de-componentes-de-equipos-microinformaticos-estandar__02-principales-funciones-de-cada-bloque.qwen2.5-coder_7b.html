Bloque Central Procesador (CPU)

El CPU es el "cérebro" del ordenador, ejecutando instrucciones y realizandoálculos.

Bloque de Memoria Principal (RAM)

  • Almacena datos temporales que el CPU necesita para su operación
  • Tamaño y velocidad son factores clave
  • Ejemplos: DDR4, DDR5

Bloque de Almacenamiento Principal (Disco Duro)

Almacena datos permanentemente y permite acceso rápido a ellos.

Bloque de Entrada/Salida (I/O)

  • Permite la comunicación entre el ordenador y el exterior
  • Ejemplos: teclado, ratón, monitor, impresora

Bloque de Alimentación (Fuente de Poder)

Proporciona energía eléctrica al sistema para su funcionamiento.

Bloque de Controlador de Placa Base

Coordina las operaciones entre los diferentes componentes del ordenador.

Caso del Ordenador

  • Protege y organiza los componentes internos
  • Ejemplos: aluminio, plástico, metálico

Cableado Interno

Conecta los diferentes componentes del ordenador entre sí.

Mini-Quiz: Identificación de bloques

  • ¿Qué componente es el "cérebro" del ordenador?
  • ¿Qué tipo de memoria almacena datos permanentemente?
  • ¿Qué bloque coordina las operaciones entre los componentes?

Buenas prácticas en la selección de componentes

  • Elegir componentes compatibles
  • Considere el rendimiento y las necesidades del usuario
  • Mantén un buen equilibrio entre costo y rendimiento

Errores comunes en la selección de componentes

  • Comprar componentes de mayor potencia que necesitas
  • No considerar el cableado interno adecuado
  • Ignorar las recomendaciones del fabricante

Comparativa de componentes

Tipo Ejemplo Rendimiento Precio

Checklist de selección de componentes

  • Especifica tus necesidades (uso, rendimiento)
  • Busca compatibilidad entre componentes
  • Compara precios y prestaciones
  • Considere la seguridad y el mantenimiento
