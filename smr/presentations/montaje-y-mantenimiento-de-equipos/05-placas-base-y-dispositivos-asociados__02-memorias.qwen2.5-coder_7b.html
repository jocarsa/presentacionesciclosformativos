Placas base y dispositivos asociados: Memorias

Objetivo: Aprender sobre las memorias en placas base y sus aspectos de montaje y mantenimiento.

Memorias en Placas Base

  • Tipos principales de memorias RAM (RAM DDR, DDR2, DDR3, DDR4)
  • Características técnicas: velocidad, capacidad y tipo
  • Importancia para el rendimiento del sistema

Instalación de Memorias RAM

  1. Desconectar la alimentación del sistema
  2. Alinear las ranuras de memoria con los orificios en el teclado
  3. Pulsar suavemente para instalar la memoria
  4. Repetir el proceso para todas las memorias necesarias

Verificación de Instalación

  • Comprobar que las memorias estén firmemente instaladas
  • Reconectar la alimentación y encender el sistema
  • Verificar los mensajes del BIOS sobre la memoria instalada

Buenas Prácticas al Instalar Memorias

  • No usar fuerza bruta para instalar las memorias
  • Seguir el orden correcto de instalación (memoria par, memoria impar)
  • Evitar contactos eléctricos con los componentes internos

Errores Comunes al Instalar Memorias

  • No instalar las memorias en ranuras incorrectas
  • No conectar adecuadamente la alimentación
  • Contactos eléctricos con otros componentes internos

Checklist de Instalación de Memorias RAM

  • Desconectar la alimentación
  • Alinear las ranuras con los orificios del teclado
  • Pulsar suavemente para instalar
  • Repetir el proceso para todas las memorias
  • Verificar la instalación

Actividad: Prueba de Rendimiento

Ejecutar pruebas de rendimiento (como Memtest86+) para verificar la correcta instalación y funcionamiento de las memorias.

Mantenimiento de las Memorias RAM

  • Evitar sobrecalentamiento
  • No exponer a luz directa
  • Limpieza periódica con un paño suave

Normativa y Seguridad

Seguir las normativas de seguridad al manejar componentes electrónicos.

  • No manipular componentes mientras el sistema esté encendido
  • Usar equipo de protección personal (gafas protectoras, guantes)
