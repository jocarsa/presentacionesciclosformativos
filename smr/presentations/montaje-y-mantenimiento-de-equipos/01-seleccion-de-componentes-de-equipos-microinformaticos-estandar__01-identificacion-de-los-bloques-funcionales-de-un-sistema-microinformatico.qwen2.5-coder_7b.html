Portada

Selección de componentes de equipos microinformáticos estándar

Objetivo: Identificar los bloques funcionales de un sistema microinformático.

Bloque Central del Sistema Microinformático

  • CPU (Procesador)
  • RAM (Memoria Volátil)
  • Cache
  • Controlador de Flujo
  • Unidad Aritmético-Lógica (UAL)

Bloque Periférico del Sistema Microinformático

  • Entrada/Salida (I/O) - Teclado, Mouse, Monitor, Impresora
  • Almacenamiento Secundario - Disco Duro, SSD
  • Batería y Fuente de Alimentación

Bloque de Control del Sistema Microinformático

  • Controlador de Procesamiento (CP)
  • Controlador de Interrupiones
  • Controlador de Memoria

Bloque de Interfaz del Sistema Microinformático

  • Interfaz de Red (NIC)
  • Interfaz USB
  • Interfaz PCI/PCIe

Bloque de Almacenamiento del Sistema Microinformático

  • Almacenamiento Volátil - RAM
  • Almacenamiento No Volátil - Disco Duro, SSD

Bloque de Seguridad del Sistema Microinformático

  • Crypto-Acelerador
  • Firewall
  • Antivirus

Bloque de Comunicación del Sistema Microinformático

  • Interfaz Serial (RS-232)
  • Interfaz Paralela (PS/2)
  • Interfaz USB

Bloque de Energía del Sistema Microinformático

  • Batería
  • Fuente de Alimentación (PSU)

Bloque de Temperatura y Refrigeración del Sistema Microinformático

  • Sensor de Temperatura
  • Fan (Ventilador)
  • Termómetro Digital

Bloque de Configuración del Sistema Microinformático

  • BIOS/UEFI
  • Configurador de BIOS

Bloque de Diagnóstico del Sistema Microinformático

  • Diagnóstico de Hardware
  • Diagnóstico de Software

Bloque de Actualización del Sistema Microinformático

  • Actualizaciones del Sistema Operativo
  • Actualizaciones del Firmware

Bloque de Seguridad del Sistema Microinformático

  • Crypto-Acelerador
  • Firewall
  • Antivirus

Bloque de Comunicación del Sistema Microinformático

  • Interfaz Serial (RS-232)
  • Interfaz Paralela (PS/2)
  • Interfaz USB

Bloque de Energía del Sistema Microinformático

  • Batería
  • Fuente de Alimentación (PSU)

Resumen

Los bloques funcionales de un sistema microinformático incluyen el bloque central, periférico, control, interfaz, almacenamiento, seguridad, comunicación, energía y temperatura.
