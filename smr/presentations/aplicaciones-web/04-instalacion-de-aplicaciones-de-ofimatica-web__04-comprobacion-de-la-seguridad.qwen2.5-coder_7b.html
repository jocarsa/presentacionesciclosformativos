Instalación de aplicaciones de ofimática web

Objetivo: Comprender los pasos para instalar y verificar la seguridad de aplicaciones de ofimática web.

Definición de aplicaciones de ofimática web

  • Son herramientas que se ejecutan en el navegador del usuario.
  • No requieren instalación local, solo conexión a internet.
  • Ejemplos: Google Docs, Microsoft Office Online, WPS Office Online.

Principales ventajas de las aplicaciones web de ofimática

  • Acceso desde cualquier dispositivo con conexión a internet.
  • No requiere instalación o mantenimiento local.
  • Compartir y colaborar en tiempo real.

Requisitos previos para instalar aplicaciones web de ofimática

  • Navegador web moderno (Chrome, Firefox, Edge).
  • Conexión a internet estable.
  • Cuenta de usuario en el servicio de la aplicación.

Paso 1: Acceso al sitio web de la aplicación

Ingresa la URL del sitio web correspondiente a la aplicación que deseas instalar (por ejemplo, https://docs.google.com).

Paso 2: Inicio de sesión

Inicia sesión con tu cuenta de usuario existente o crea una nueva si es necesario.

Paso 3: Creación de un nuevo documento

Una vez iniciada la sesión, selecciona el tipo de documento que deseas crear (por ejemplo, hoja deálculo, presentación).

Paso 4: Comenzar a trabajar

Ahora puedes comenzar a editar y trabajar en tu documento directamente desde el navegador.

Comprobación de la seguridad en aplicaciones web de ofimática

  • Verifica que el sitio web utilice HTTPS (la URL comience con https://).
  • Cuida tu privacidad al no compartir información sensible.
  • No descargues archivos desconocidos desde la aplicación.

Uso de contraseñas fuertes

Para mejorar la seguridad, utiliza contraseñas complejas y únicas para cada cuenta.

Actividad: Verificar la seguridad en una aplicación web de ofimática

Elige una aplicación web de ofimática y verifica si cumple con los criterios de seguridad mencionados.

Caso práctico: Seguridad en Google Docs

Google Docs utiliza HTTPS y ofrece opciones para compartir documentos de manera segura. Verifica estos aspectos en tu cuenta.

Buenas prácticas adicionales

  • No guardar archivos en la computadora local a menos que sea necesario.
  • Usa el modo de edición colaborativa con precaución.
  • Mantén actualizadas las aplicaciones web para aprovechar nuevas funcionalidades y mejoras de seguridad.

Errores comunes al instalar aplicaciones web de ofimática

  • Ingresar la URL incorrecta del sitio web.
  • No iniciar sesión con una cuenta válida.
  • No utilizar contraseñas seguras.

Mini-quiz: Seguridad en aplicaciones web de ofimática

Pregunta 1: ¿Qué significa HTTPS?
Respuesta: Protocolo seguro para la comunicación entre el navegador y el servidor.

Checklist de seguridad en aplicaciones web de ofimática

  • Verifica la URL con HTTPS.
  • Crea contraseñas seguras.
  • No descargas archivos desconocidos.
  • Mantén las aplicaciones actualizadas.

Resumen

Para instalar y usar aplicaciones web de ofimática, sigue los pasos básicos y asegúrate de verificar la seguridad para proteger tus datos.

Normativa general sobre privacidad en línea

  • Datos personales: Protegidos por leyes como el RGPD (Europa) y la Ley Orgánica 34/1998 (España).
  • Privacidad online: Importancia de usar aplicaciones web seguras.
