Instalación y Utilización de Aplicaciones Web de Ofimática

Objetivo: Aprender a instalar y utilizar aplicaciones web de ofimática como Google Docs, Sheets y Slides.

Introducción

  • Definición de aplicaciones web de ofimática
  • Ventajas de usar aplicaciones web en ofimática
  • Ejemplos populares: Google Docs, Sheets y Slides

Requisitos Previos

  • Conexión a Internet estable
  • Cuenta de correo electrónico (Google)
  • Navegador web moderno (Chrome, Firefox, Edge)

Instalación de Google Docs

  1. Ingresar a docs.google.com
  2. Identificarse con su cuenta de correo electrónico
  3. Clic en "Crear" para crear un nuevo documento

Instalación de Google Sheets

  1. Ingresar a sheets.google.com
  2. Identificarse con su cuenta de correo electrónico
  3. Clic en "Crear" para crear una nueva hoja deálculo

Instalación de Google Slides

  1. Ingresar a slides.google.com
  2. Identificarse con su cuenta de correo electrónico
  3. Clic en "Crear" para crear una nueva presentación

Interfaz de Google Docs

  • Menú superior con opciones como Archivo, Edición, Formatos y Herramientas
  • Barras laterales para formato y herramientas adicionales
  • Vista previa en la parte inferior

Interfaz de Google Sheets

  • Celdas numeradas y letras para datos
  • Fórmulas eneldas paraálculos automáticos
  • Herramientas de formato y análisis en la parte superior
  • Gráficos y tablas dinámicas

Interfaz de Google Slides

  • Pantalla de edición con herramientas de diseño
  • Biblioteca de plantillas y imágenes
  • Herramientas de transición entre diapositivas
  • Exportación a formatos como PDF o PowerPoint

Funcionalidades Básicas de Google Docs

  • Escribir y editar texto
  • Copiar, cortar y pegar contenido
  • Realizar búsquedas y reemplazos
  • Formato de texto (negrita, cursiva, etc.)

Funcionalidades Básicas de Google Sheets

  • Ingresar datos eneldas
  • Cálculos básicos y fórmulas avanzadas
  • Filtros y ordenamiento de datos
  • Condicionales (formatos condicionales)

Funcionalidades Básicas de Google Slides

  • Añadir texto y imágenes
  • Diseño de diapositivas (plantillas, diseño personalizado)
  • Transiciones entre diapositivas
  • Exportación a formatos como PDF o PowerPoint

Trabajo en Equipo con Google Docs

  • Compartir documentos con otros usuarios
  • Revisar y aprobar cambios
  • Historial de versiones

Trabajo en Equipo con Google Sheets

  • Compartir hojas deálculo con otros usuarios
  • Revisar y aprobar cambios
  • Historial de versiones
  • Funciones colaborativas (comentarios, sugerencias)

Trabajo en Equipo con Google Slides

  • Compartir presentaciones con otros usuarios
  • Revisar y aprobar cambios
  • Historial de versiones
  • Funciones colaborativas (comentarios, sugerencias)

Buenas Prácticas

  • Guardar documentos regularmente
  • Usar nombres descriptivos para archivos
  • Evitar sobrescribir versiones antiguas
  • Compartir con permisos adecuados

Errores Comunes

  • No guardar cambios antes de cerrar el navegador
  • Compartir archivos sin revisión
  • Perder acceso a documentos compartidos
  • Usar nombres genéricos para archivos
