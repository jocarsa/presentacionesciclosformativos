Instalación de aplicaciones ofimáticas

Objetivo: Comprender los tipos de aplicaciones ofimáticas y cómo instalarlas.

Tipos de aplicaciones ofimáticas

  • Sofware de escritorio: Instalado directamente en el sistema operativo.
  • Servicios web: Accesibles a través del navegador web.
  • Aplicaciones móviles: Disponibles para dispositivos Android y iOS.

Ejemplos de software de escritorio

  • Microsoft Office (Word, Excel, PowerPoint)
  • LibreOffice (Calc, Writer, Impress)
  • Google Docs, Sheets, Slides

Ejemplos de servicios web ofimáticos

  • Google Workspace (G Suite)
  • Microsoft 365 Online
  • Zoho Office

Ejemplos de aplicaciones móviles ofimáticas

  • Google Docs, Sheets, Slides (Android/iOS)
  • Microsoft Word, Excel, PowerPoint (Android/iOS)
  • LibreOffice Mobile (Android/iOS)

Caso: Instalación de Microsoft Office en Windows

Pasos básicos para instalar Microsoft Office en un sistema operativo Windows.

Proceso: Instalación de Google Docs en navegador web

  1. Ingresar a docs.google.com
  2. Crear una cuenta si es necesario
  3. Comenzar a usar las aplicaciones directamente desde el navegador

Buenas prácticas al instalar aplicaciones ofimáticas

  • Verificar la fuente de descarga (oficial, confiable)
  • Aceptar las condiciones de uso y licencias
  • No instalar plugins o complementos no reconocidos

Errores comunes al instalar aplicaciones ofimáticas

  • Instalar software de manera incorrecta (mal archivo, versión incompatibles)
  • No tener permisos de administrador
  • No aceptar actualizaciones importantes

Comparativa: Software vs. Servicios web vs. Aplicaciones móviles

Tipo de aplicación Ventajas Inconvenientes
Software de escritorio Instalación local, offline Puede requerir actualizaciones manuales
Servicios web No requiere instalación, acceso desde cualquier lugar Puede requerir conexión a internet
Aplicaciones móviles Acceso en dispositivos móviles, offline (con sincronización) Puede requerir permisos adicionales y actualizaciones frecuentes
