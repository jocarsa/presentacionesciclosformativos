Instalación de aplicaciones

Objetivo: Aprender los pasos para instalar paquetes ofimáticos en sistemas operativos Windows y Linux.

Paquetes ofimáticos

Definición: Conjunto de aplicaciones que permiten la creación, edición y gestión de documentos electrónicos.

Sistemas operativos compatibles

  • Windows (Microsoft Office)
  • Linux (LibreOffice)

Microsoft Office

Paquete de aplicaciones ofimáticas desarrollado por Microsoft.

LibreOffice

Alternativa gratuita y de código abierto a Microsoft Office.

Pasos para instalar Microsoft Office

  1. Descargar el instalador desde el sitio web oficial de Microsoft.
  2. Ejecutar el instalador y seguir las instrucciones.
  3. Aceptar los términos y condiciones de licencia.
  4. Seleccionar la versión adecuada para su sistema operativo.
  5. Completar la instalación siguiendo los pasos indicados.

Pasos para instalar LibreOffice

  1. Descargar el instalador desde el sitio web oficial de LibreOffice.
  2. Ejecutar el instalador y seguir las instrucciones.
  3. Aceptar los términos y condiciones de licencia.
  4. Seleccionar la versión adecuada para su sistema operativo.
  5. Completar la instalación siguiendo los pasos indicados.

Buenas prácticas de instalación

  • Ejecutar el instalador como administrador.
  • Desinstalar aplicaciones no utilizadas antes de instalar nuevas.
  • No seleccionar opciones adicionales innecesarias durante la instalación.

Errores comunes en la instalación

  • Sin conexión a Internet durante el proceso de instalación.
  • No aceptar los términos y condiciones de licencia.
  • Ejecutar el instalador como usuario no administrador.

Actividad: Instalación práctica

Instalar Microsoft Office en un sistema operativo Windows y LibreOffice en un sistema operativo Linux.

Resumen de la instalación de paquetes ofimáticos

  • Microsoft Office: Paquete comercial con soporte al cliente limitado.
  • LibreOffice: Alternativa gratuita y de código abierto con una comunidad activa en línea.
  • Pasos para la instalación: Descargar, ejecutar el instalador, aceptar términos, seleccionar versión y completar proceso.

Normativa de software

Normativas como la Ley General de Protección de Datos (LGPD) requieren el uso de software que cumpla con ciertos estándares de privacidad y seguridad.

Caso práctico: Cumplimiento normativo

Una empresa debe instalar un paquete ofimático que cumpla con la LGPD para proteger los datos personales de sus empleados y clientes.

Cierre

Recuerde practicar las buenas prácticas de instalación y mantenerse al día con las normativas de software.
