Objetivo: Comprender los tipos de licencias software y su aplicación en la instalación.

Tipos de licencias software

  • Licencia de uso único (Single User License)
  • Licencia por usuario (User License)
  • Licencia por dispositivo (Device License)
  • Licencia por sitio web (Site License)
  • Licencia por servidor (Server License)

Tipos de licencias software

Licencia de uso único (Single User License): Permite la instalación y uso en un solo computador.

Tipos de licencias software

Licencia por usuario (User License): Permite la instalación y uso para un número específico de usuarios.

Tipos de licencias software

Licencia por dispositivo (Device License): Permite la instalación en un número específico de dispositivos.

Tipos de licencias software

Licencia por sitio web (Site License): Permite el uso en todos los computadores dentro de una misma red o sitio web.

Tipos de licencias software

Licencia por servidor (Server License): Permite la instalación y uso en un servidor específico.

Ejemplo: Licencia de uso único

Aplicación: Microsoft Office Home and Business 365

Características: Instalable en un solo computador, acceso a todos los productos de la suite.

Ejemplo: Licencia por usuario

Aplicación: Adobe Creative Suite

Características: Instalable en un número específico de computadores, acceso a todos los productos.

Ejemplo: Licencia por dispositivo

Aplicación: Autodesk AutoCAD

Características: Instalable en un número específico de dispositivos, acceso a todos los productos.

Ejemplo: Licencia por sitio web

Aplicación: Google Workspace for Education

Características: Acceso a todos los productos desde cualquier computador dentro de la red escolar.

Ejemplo: Licencia por servidor

Aplicación: Microsoft Exchange Server

Características: Instalable en un servidor específico, gestión de correo electrónico corporativo.

Caso: Implementación de licencias por usuario

Empresa XYZ implementa Adobe Creative Suite para sus diseñadores.ada diseñador recibe una licencia por usuario.

Caso: Implementación de licencias por dispositivo

Instituto ABC adquiere Autodesk AutoCAD para su departamento de ingeniería. Instalan el software en 10 computadoras.

Proceso: Compra y instalación de software con licencia por usuario

  1. Comprar la licencia por usuario
  2. Instalar el software en cada computador asignado
  3. Registrar la licencia en el sistema del software
  4. Asignar los permisos de uso según las políticas internas

Buenas prácticas: Manejo de licencias por dispositivo

  • No compartir licencias entre dispositivos
  • Registrar todas las instalaciones en el sistema del software
  • Mantener un inventario actualizado de los dispositivos con licencias asignadas

Buenas prácticas: Manejo de licencias por usuario

  • Asignar licencias según las necesidades del usuario
  • No compartir licencias entre usuarios
  • Mantener un inventario actualizado de los usuarios con licencias asignadas

Errores comunes: Instalación en computadores no autorizados

Instalar software con licencia por dispositivo en computadoras fuera del alcance de la empresa.

Errores comunes: Compartir licencias entre usuarios

Compartir licencias por usuario entre diferentes usuarios dentro de la organización.
