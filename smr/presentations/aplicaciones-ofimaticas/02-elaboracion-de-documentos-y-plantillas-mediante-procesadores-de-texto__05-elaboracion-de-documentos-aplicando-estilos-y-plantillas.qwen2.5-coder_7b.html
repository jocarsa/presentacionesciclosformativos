Elaboración de documentos y plantillas mediante procesadores de texto

Objetivo: Aprender a crear documentos profesionales utilizando estilos y plantillas en procesadores de texto.

Introducción a los Procesadores de Texto

  • Microsoft Word
  • Google Docs
  • LibreOffice Writer
  • LaTeX

Concepto de Estilos en Procesadores de Texto

Estilos predefinidos que permiten formatear texto automáticamente.

Ejemplo de Aplicación de Estilos

  • Títulos (H1, H2, etc.)
  • Subtítulos
  • Cuerpo del texto
  • Listas ordenadas y no ordenadas
  • Enlaces y citas

Plantillas en Procesadores de Texto

Modelos predefinidos que facilitan la creación de documentos estándar.

Ejemplo de Plantilla

  • Cartas profesionales
  • Reportes financieros
  • Presentaciones
  • Formularios

Caso Práctico: Creación de una Carta Profesional

Pasos para crear una carta utilizando estilos y plantillas.

Proceso de Creación de un Documento

  1. Elegir el procesador de texto adecuado
  2. Seleccionar o crear una plantilla
  3. Ajustar los estilos según las necesidades
  4. Ingresar el contenido del documento
  5. Revisar y formatear

Buenas Prácticas en la Creación de Documentos

  • Consistencia en el estilo
  • Uso correcto del formato
  • Revisión y corrección ortográfica
  • Actualización regular

Errores Comunes en la Creación de Documentos

  • Formato inconsistente
  • Errores ortográficos y gramaticales
  • Ausencia de información relevante
  • No actualización del contenido

Comparativa entre Procesadores de Texto
