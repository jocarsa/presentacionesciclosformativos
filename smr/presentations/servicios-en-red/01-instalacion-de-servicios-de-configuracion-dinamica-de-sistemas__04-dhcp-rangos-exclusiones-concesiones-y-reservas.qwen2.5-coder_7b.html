DHCP: Rangos, Exclusiones, Concesiones y Reservas

El Protocolo de Configuración Dinámica de Hosts (DHCP) asigna automáticamente direcciones IP a los dispositivos en una red.

Rangos DHCP

  • Determinan el rango de IPs disponibles para asignación dinámica.
  • Evita conflictos entre direcciones IP estáticas y dinámicas.
  • Ejemplo: 192.168.1.10 - 192.168.1.100

Exclusiones DHCP

  • Direcciones IP que no se asignan dinámicamente.
  • Usadas para dispositivos con configuración estática o reservas.
  • Ejemplo: 192.168.1.5, 192.168.1.10

Concesiones DHCP

  • Direcciones IP asignadas a dispositivos por un tiempo determinado.
  • Permiten la renovación de direcciones IP si el dispositivo está en línea.
  • Ejemplo: 192.168.1.20, con duración de 24 horas

Reservas DHCP

  • Direcciones IP asignadas permanentemente a dispositivos específicos.
  • Evita la pérdida de configuración si el dispositivo se reemplaza.
  • Ejemplo: Dirección 192.168.1.30 para un servidor web

Caso Práctico

Una empresa necesita asignar direcciones IP a sus empleados. El administrador configura DHCP con el rango 192.168.1.10-192.168.1.50, excluye las primeras 5 y últimas 5 IPs, y reserva la dirección 192.168.1.30 para el servidor web.

Proceso de Configuración DHCP

  1. Configurar el rango de direcciones IP disponibles.
  2. Excluir las direcciones IP que no se usarán dinámicamente.
  3. Reservar direcciones IP para dispositivos específicos.
  4. Reiniciar el servicio DHCP para aplicar los cambios.

Buenas Prácticas

  • Asignar rangos de IPs suficientemente grandes para evitar agotamiento.
  • Excluir las primeras y últimas direcciones del rango para gestión interna.
  • Reservar direcciones IP para dispositivos críticos como servidores.

Errores Comunes

  • No asignar suficientes direcciones IP en el rango.
  • Excluir todas las primeras y últimas direcciones, limitando la gestión interna.
  • No reservar direcciones IP para dispositivos críticos.

Comparativa con DHCP Estático

Dinámico (DHCP) Estatico
Asignación automática de direcciones IP. Configuración manual de direcciones IP.
No requiere intervención continua del administrador. Necesita intervención continua para asignar y mantener las direcciones IP.

Actividad

Resumen

  • DHCP asigna automáticamente direcciones IP a los dispositivos en una red.
  • Configurar rangos, exclusiones, concesiones y reservas es crucial para la gestión de la red.
