Dirección IP, máscara de red, puerta de enlace

Unidad: Instalación de servicios de configuración dinámica de sistemas
Documento: Servicios en red.md

Objetivos: comprender los conceptos clave y su aplicación práctica.

Dirección IP

La dirección IP (Internet Protocol) es un identificador único asignado a cada dispositivo conectado a una red.

Máscara de Red

La máscara de red define la parte fija y variable de la dirección IP, separando la red de la interfaz de red.

Puerta de Enlace

La puerta de enlace es el punto de conexión entre dos redes diferentes, permitiendo la comunicación entre ellas.

Ejemplo: Dirección IP y Máscara

  • Dirección IP: 192.168.1.100
  • Máscara de Red: 255.255.255.0
  • Red: 192.168.1.0
  • Interfaz: 192.168.1.100

Ejemplo: Puerta de Enlace

Puerta de enlace predeterminada: 192.168.1.1

Caso Práctico

Configurar la dirección IP, máscara y puerta de enlace para una nueva computadora.

Proceso: Configuración de Dirección IP

  1. Ingresar a la configuración de red del sistema operativo.
  2. Elegir la interfaz de red deseada.
  3. Seleccionar el método de asignación automática (DHCP) o manual.
  4. Si es manual, ingresar dirección IP, máscara y puerta de enlace.

Buenas Prácticas

  • No compartir la misma dirección IP con otros dispositivos.
  • Configurar una puerta de enlace correcta para acceso a Internet.
  • Mantener las máscaras de red consistentes entre dispositivos.

Errores Comunes

  • Configurar la misma dirección IP en dos dispositivos.
  • No configurar una puerta de enlace correcta.
  • Mixtar máscaras de red incompatibles entre dispositivos.

  • Máscara de Red: Define la parte fija y variable de la dirección IP.
  • Puerta de Enlace: Punto de conexión entre redes diferentes.
