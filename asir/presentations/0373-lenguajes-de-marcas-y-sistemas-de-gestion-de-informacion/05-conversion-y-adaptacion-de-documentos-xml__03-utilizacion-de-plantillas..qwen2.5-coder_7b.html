Conversión y adaptación de documentos

Objetivo: Aprender a utilizar plantillas para la conversión y adaptación de documentos.

¿Qué son las plantillas?

  • Plantillas son documentos que definen cómo se debe estructurar el contenido final.
  • Permiten la reutilización de código y facilitan la adaptación del contenido a diferentes formatos.
  • Son utilizadas en procesadores de plantillas como XSLT, XSL-FO o XQuery.

Ejemplo básico de plantilla XSLT

<? version="1.0" encoding="UTF-8"?>
<xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">
  <xsl:output method="" indent="yes"/>
  <xsl:template match="/libro">
    <>
      <head><title>Libro</title></head>
      <body>
        <h1>Título: <xsl:value-of select="titulo"/></h1>
        <p>Autor: <xsl:value-of select="autor"/></p>
      </body>
    </>
  </xsl:template>
</xsl:stylesheet>

Proceso de conversión con plantillas

  1. Cargar el documento fuente.
  2. Cargar la plantilla XSLT.
  3. Ejecutar el procesador de plantillas, que transforma el en el formato deseado.
  4. Obtener el resultado transformado.

Buenas prácticas al usar plantillas

  • Mantener la separación entre contenido y presentación.
  • Utilizar variables y parámetros para mayor flexibilidad.
  • Optimizar el rendimiento mediante la minimización de procesos innecesarios.

Errores comunes al usar plantillas

  • No definir correctamente las rutas XPath.
  • No gestionar adecuadamente los espacios de nombres.
  • No optimizar el procesamiento para grandes documentos.

Comparativa: XSLT vs. XSL-FO

XSLT XSL-FO
Procesa datos para generar cualquier tipo de contenido. Específico para la generación de documentos físicos (PDF, PostScript).
Más versátil pero más complejo. Menos versátil pero más fácil de aprender y usar.
