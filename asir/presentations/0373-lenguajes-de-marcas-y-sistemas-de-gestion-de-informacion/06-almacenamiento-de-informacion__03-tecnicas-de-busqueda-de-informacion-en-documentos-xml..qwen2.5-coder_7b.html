Técnicas de búsqueda de información en documentos

Objetivo: Comprender las técnicas utilizadas para buscar y recuperar información en documentos.

Introducción a los documentos

  • Estructura basada en etiquetas jerárquicas.
  • Formato de texto legible y fácilmente procesable por computadora.
  • Utilizado ampliamente para interambiar datos entre sistemas.

Tipos de búsquedas en

  • Búsqueda por etiquetas: Identificar elementos específicos.
  • Búsqueda por atributos: Filtrar elementos según sus atributos.
  • Búsqueda por contenido: Buscar texto dentro de los elementos.

Uso de XPath para búsqueda

XPath (eXtensible Path Language) es un lenguaje utilizado para navegar y seleccionar nodos en documentos.

Ejemplo básico de XPath

/libro/título/text()

Selecciona el texto del elemento "título" dentro del elemento "libro".

Búsqueda condicional con XPath

/libro[título='El Quijote']/autor/text()

Selecciona el autor del librouyo título es "El Quijote".

Búsqueda por atributos con XPath

/libro[@categoria='Ficción']/título/text()

Selecciona los títulos de todos los libros que pertenecen a la categoría "Ficción".

Búsqueda recursiva con XPath

/libro/*

Selecciona todos los elementos hijos de los elementos "libro".

Uso de XQuery para búsqueda avanzada

XQuery es un lenguaje de consulta y manipulación de datos basado en XPath.

Ejemplo básico de XQuery

for $libro in doc('biblioteca.xml')/libros/libro
where $libro/categoría = 'Ficción'
return $libro/título/text()

Selecciona los títulos de todos los libros que pertenecen a la categoría "Ficción".

Búsqueda por contenido con XQuery

for $libro in doc('biblioteca.xml')/libros/libro
where contains($libro/título, 'El')
return $libro/título/text()

Selecciona los títulos que contienen la palabra "El".

Búsqueda por atributo con XQuery

for $libro in doc('biblioteca.xml')/libros/libro
where $libro/@categoria = 'Ficción'
return $libro/título/text()

Selecciona los títulos de todos los libros que pertenecen a la categoría "Ficción".

Búsqueda recursiva con XQuery

for $elemento in doc('biblioteca.xml')//*
return $elemento/name()

Selecciona el nombre de todos los elementos en el documento.

Buenas prácticas en búsqueda de información

  • Utilizar XPath o XQuery según la complejidad de la consulta.
  • Optimizar las consultas para mejorar el rendimiento.
  • Validar los documentos antes de realizar búsquedas.

Caso práctico: Búsqueda en un sistema de gestión de bibliotecas

Un sistema de gestión de bibliotecas utiliza para almacenar información sobre los libros. Se requiere una búsqueda rápida y precisa de títulos y autores.

Ejercicio: Crear una consulta XPath

Crear una consulta XPath que seleccione todos los títulos de los libros en la categoría "Non-Ficción" con más de 50 páginas.

Chequeo de errores comunes en búsqueda

  • Olvidar el uso del prefijo "ns:" si se utiliza un espacio de nombres.
  • No considerar la distinción entre mayúsculas y minúsculas en los atributos o contenido.
  • No validar correctamente los documentos antes de realizar búsquedas.

Resumen

Las técnicas de búsqueda en documentos son esenciales para el acceso y recuperación de información. XPath y XQuery ofrecen herramientas poderosas para realizar consultas complejas.

Normativa relevante

  • ISO/IE 9075:2016 - SQL:2016 (para consultas en bases de datos relacionales).
  • W3 Schema Definition Language (XSD) para validar documentos.
