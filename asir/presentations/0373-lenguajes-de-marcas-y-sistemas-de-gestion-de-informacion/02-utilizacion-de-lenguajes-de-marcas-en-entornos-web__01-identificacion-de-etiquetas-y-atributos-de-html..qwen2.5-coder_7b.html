Identificación de etiquetas y atributos de

Objetivo: Comprender las etiquetas y atributos fundamentales de para la creación de páginas web.

¿Qué son las etiquetas en?

  • Etiquetas definen el tipo de contenido que se está insertando.
  • Son delimitadas por corchetes angulares (< >).
  • Ejemplo: <p>Este es un párrafo.</p>

¿Qué son los atributos en?

  • Atributos proporcionan información adicional sobre las etiquetas.
  • Son escritos dentro de la etiqueta, antes delierre.
  • Ejemplo: <a href="https://www.example.com">Enlace</a>

Etiquetas básicas de

  • <p> (Parágrafo)
  • <a> (Enlace)
  • <img> (Imagen)
  • <div> (División)
  • <ul> (Lista no ordenada)

Atributos comunes de

  • id: Identifica un elemento único.
  • class: Define una o varias clases para estilos CSS.
  • src: Especifica la ruta del archivo multimedia.
  • href: Define el enlace de destino.
  • alt: Proporciona texto alternativo para imágenes.

Ejemplo de uso de etiquetas y atributos

<img src="logo.png" alt="Logo de la empresa">

Este código inserta una imagen con el texto alternativo "Logo de la empresa".

Buenas prácticas al usar etiquetas y atributos

  • Usar etiquetas semánticas.
  • Atribuir id únicos a elementos.
  • Utilizar class para estilos CSS.
  • Completar atributos obligatorios.
  • Mantener el código limpio y legible.

Errores comunes al usar etiquetas y atributos

  • No cerrar las etiquetas.
  • Usar etiquetas obsoletas.
  • Omitir atributos obligatorios.
  • Combinar etiquetas incorrectamente.
  • No usar id únicos.

Resumen: Etiquetas y atributos de

Etiquetas definen el tipo de contenido, mientras que los atributos proporcionan información adicional. Es importante usar etiquetas semánticas, atribuir id únicos, utilizar class para estilos CSS, completar atributos obligatorios y mantener el código limpio y legible.
