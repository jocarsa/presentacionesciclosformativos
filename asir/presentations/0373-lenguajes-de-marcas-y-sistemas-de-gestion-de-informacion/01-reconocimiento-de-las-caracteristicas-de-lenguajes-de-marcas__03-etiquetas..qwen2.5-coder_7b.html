Objetivo: Comprender las etiquetas en lenguajes de marcas y su importancia.

¿Qué son las etiquetas?

Etiquetas son palabras clave que definen el significado y la estructura del contenido en un documento.

Estructura básica de una etiqueta

  • <etiqueta> (Abrir)
  • Contenido
  • </etiqueta> (Cerrar)

Ejemplo de etiqueta

<p>Este es un párrafo.</p>

Etiquetas semánticas

  • <header>
  • <footer>
  • <article>
  • <section>

Etiquetas de encabezado

  • <h1> a <h6>
  • Ejemplo: <h1>Título principal</h1>

Etiquetas de enlace

<a href="https://www.example.com">Enlace a ejemplo</a>

Etiquetas de imagen

<img src="imagen.jpg" alt="Descripción de la imagen">

Etiquetas de lista

  • <ul> (Desordenada)
  • <ol> (Ordenada)
  • <li> (Elemento de lista)

Etiquetas de formulario

<form action="/submit" method="post">
  <input type="text" name="nombre">
  <button type="submit">Enviar</button>
</form>

Etiquetas de tablas

<table>
  <tr>
    <th>Encabezado 1</th>
    <th>Encabezado 2</th>
  </tr>
  <tr>
    <td>Dato 1</td>
    <td>Dato 2</td>
  </tr>
</table>

Etiquetas de estilo

  • <style>
  • <link rel="stylesheet" href="estilos.css">

Etiquetas de metadatos

<meta name="keywords" content=", CSS,">
<meta name="description" content="Descripción del sitio">

Etiquetas de multimedia

  • <video>
  • <audio>

Etiquetas de cita

<q>ita corta</q>
<blockquote>ita larga</blockquote>

Etiquetas de código

<code><p>Este es un párrafo.</p></code>

Mini-quiz

  1. ¿Qué etiqueta se usa para crear una lista ordenada?
  2. ¿Cuál es la diferencia entre <div> y <section>?
  3. ¿Qué hace la etiqueta <a>?

Checklist de buenas prácticas

  • Usar etiquetas semánticas
  • Añadir atributos como alt a las imágenes
  • Cerrar todas las etiquetas
  • Evitar etiquetas obsoletas

Errores comunes

  • No cerrar etiquetas
  • Usar etiquetas incorrectas
  • Atributos mal escritos o ausentes

Comparativa de etiquetas

Etiqueta Función
<p> Párrafo
<h1> Encabezado principal
<a> Enlace

Actividad práctica

Crea un documento con las siguientes etiquetas: <header>, <nav>, <main>, <section>, y <footer>.

Resumen

  • Etiquetas definen la estructura del contenido en.
  • Existen etiquetas semánticas, de encabezado, enlace, imagen, lista, formulario, tabla, estilo, metadatos y multimedia.
  • Buenas prácticas incluyen usar etiquetas semánticas, cerrar todas las etiquetas y evitar errores comunes.

Normativa

W3: Organización encargada de establecer estándares para la web. Recomienda el uso de etiquetas semánticas y mejores prácticas.
