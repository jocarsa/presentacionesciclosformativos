Estructuras del lenguaje en scripting para sistemas operativos

Objetivo: Comprender las estructuras fundamentales de los lenguajes de scripting utilizados en sistemas operativos.

Introducción a los lenguajes de scripting

  • Lenguajes como,, PowerShell y Ruby son esenciales para automatización y gestión de sistemas operativos.
  • Permiten la creación de scripts que pueden ejecutar tareas repetitivas o complejas de manera eficiente.

Sintaxis básica de

Bash es el shell predeterminado en sistemas operativos Linux y macOS.

echo "Hola, mundo!"

Estructuras condicionales en

  • if-else: Permite ejecutar bloques de código según la evaluación de una expresión booleana.
  • ejemplo: 
    if [ "$USER" == "root" ]; then
                echo "Eres root"
            else
                echo "No eres root"
            fi

Bucles en

  • for: Ejecuta un bloque de código para cada elemento de una lista.
  • while: Repite un bloque de código mientras se cumpla una condición.
  • ejemplo: 
    for i in 1 2 3; do
                echo "Iteración $i"
            done

Estructuras de control en

es un lenguaje de scripting popular para sistemas operativos.

  • if-elif-else: Permite tomar decisiones basadas en múltiples condiciones.
  • for y while: Permiten iterar sobre elementos o mientras se cumpla una condición.

Ejemplo de script

if __name__ == "__main__":
    print("Este es el punto de entrada del programa")
    for i in range(3):
        print(f"Iteración {i}")

Estructuras condicionales en PowerShell

PowerShell es un lenguaje de scripting para sistemas operativos Windows.

  • if, elseif, else: Permite ejecutar bloques de código según la evaluación de una expresión booleana.
  • ejemplo: 
    if ($isAdmin) {
                Write-Output "Eres administrador"
            } else {
                Write-Output "No eres administrador"
            }

Bucles en PowerShell

  • for: Ejecuta un bloque de código para cada elemento de una lista.
  • while: Repite un bloque de código mientras se cumpla una condición.
  • ejemplo: 
    for ($i = 1; $i -le 3; $i++) {
                Write-Output "Iteración $i"
            }

Estructuras de control en Ruby

Ruby es un lenguaje de scripting versátil y fácil de aprender.

  • if, elsif, else: Permite tomar decisiones basadas en múltiples condiciones.
  • for y while: Permiten iterar sobre elementos o mientras se cumpla una condición.

Ejemplo de script Ruby

if ARGV[0] == 'admin'
        puts "Eres administrador"
    else
        puts "No eres administrador"
    end

    (1..3).each do |i|
        puts "Iteración #{i}"
    end

Variables en, y Ruby

  • Bash: Se declaran con el nombre de la variable seguido del valor.
  • : Se declaran con el nombre de la variable seguido del valor, sin necesidad de tipo explícito.
  • Ruby: Se declaran con el nombre de la variable seguido del valor, sin necesidad de tipo explícito.

Funciones en, y Ruby

  • Bash: Se definen usando la palabra clave function o simplemente el nombre seguido del código.
  • : Se definen usando la palabra clave def seguida del nombre de la función y los parámetros.
  • Ruby: Se definen usando la palabra clave def seguida del nombre de la función y los parámetros.

Ejemplo de función en

function greet {
        echo "Hola, $1!"
    }

    greet "Mundo"

Ejemplo de función en

def greet(name):
        return f"Hola, {name}!"

    print(greet("Mundo"))

Ejemplo de función en Ruby

def greet(name)
        "Hola, #{name}!"
    end

    puts greet("Mundo")

Manejo de errores en

  • set -e: Salir del script si ocurre un error.
  • trap 'echo "Error"' ERR: Ejecutar una acción cuando ocurra un error.

Manejo de errores en

  • try-except: Permite capturar y manejar excepciones.
  • finally: Ejecutar código después de un bloque try-except, independientemente de si ocurrió una excepción o no.

Manejo de errores en Ruby

  • begin-rescue: Permite capturar y manejar excepciones.
  • ensure: Ejecutar código después de un bloque begin-rescue, independientemente de si ocurrió una excepción o no.

Resumen

  • Bash,, PowerShell y Ruby son lenguajes de scripting utilizados en sistemas operativos.
  • Cada uno tiene sus propias estructuras de control y manejo de errores.
  • Comprender estas estructuras es crucial para la automatización y gestión de sistemas operativos.
