Administración de servidores de impresión

Objetivo: Comprender las órdenes para la gestión de impresoras y trabajos en sistemas operativos.

Introducción a la administración de impresoras

  • Definición de impresión y servidores de impresión.
  • Importancia de la gestión eficiente de impresoras en entornos corporativos.

Comandos básicos para gestionar impresoras

  • lpstat -p: Muestra el estado de las impresoras.
  • lpstat -o: Muestra los trabajos en cola.
  • lpr -P nombre_impresora archivo: Envía un trabajo a una impresora específica.

Comandos para gestionar trabajos de impresión

  • cancel trabajo_id: Cancela un trabajo específico.
  • lprm -P nombre_impresora trabajo_id: Elimina un trabajo en cola de una impresora específica.
  • lpq -P nombre_impresora: Muestra la cola de trabajos de una impresora específica.

Buenas prácticas para la gestión de impresoras

  • Monitorear regularmente el estado de las impresoras y sus colas.
  • Asignar prioridades a los trabajos según su urgencia.
  • Mantener actualizados los drivers e impresoras.

Errores comunes en la gestión de impresoras

  • Olvidar el nombre de la impresora al enviar un trabajo.
  • No cancelar trabajos en cola cuando ya no son necesarios.
  • No mantener actualizados los drivers y firmware de las impresoras.

Comando para agregar una nueva impresora

lpadmin -p nombre_impresora -v uri_de_impresora -E

Comando para eliminar una impresora

lpadmin -x nombre_impresora

Comando para cambiar la prioridad de un trabajo

lpr -P nombre_impresora -o job-priority=nivel archivo

Comando para enviar trabajos a una impresora por defecto

lp archivo

Comando para ver la configuración de una impresora

lpstat -p nombre_impresora -l

Comando para enviar trabajos a múltiples impresoras

lp -d impresora1,impresora2 archivo

Comando para ver la lista de impresoras disponibles

lpstat -a

Comando para enviar trabajos en formato PDF

lpr -P nombre_impresora archivo.pdf

Comando para enviar trabajos en formato PostScript

lpr -P nombre_impresora archivo.ps
