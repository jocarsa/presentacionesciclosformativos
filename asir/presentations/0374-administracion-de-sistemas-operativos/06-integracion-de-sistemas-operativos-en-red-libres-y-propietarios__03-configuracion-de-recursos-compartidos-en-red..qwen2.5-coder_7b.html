Portada

Configuración de recursos compartidos en red

Objetivo: Aprender cómo configurar y administrar recursos compartidos en redes tanto libres como propietarias.

Concepto de Recursos Compartidos

Los recursos compartidos en red son archivos, impresoras, discos duros o cualquier otro recurso que puede ser accedido por varios usuarios simultáneamente.

Ejemplo de Configuración

  • Windows: Compartir una carpeta mediante el explorador de archivos.
  • Linux: Usar el comando 'smbd' y 'nmbd' para compartir recursos.

Caso Práctico

Configurar una impresora compartida en una red Windows.

Proceso de Configuración

  1. Instalar el controlador de la impresora en el servidor.
  2. Compartir la impresora desde el explorador de archivos.
  3. Añadir la impresora compartida desde los controladores de impresión del cliente.

Buenas Prácticas

  • Usar nombres descriptivos para recursos compartidos.
  • Configurar permisos adecuados para evitar accesos no autorizados.
  • Mantener actualizadas las versiones del software de red y seguridad.

Errores Comunes

  • No configurar correctamente los permisos de acceso.
  • No compartir el recurso con el nombre correcto.
  • No instalar el controlador del dispositivo en el servidor.

Comparativa Sistemas Operativos

Sistema Operativo Método de Configuración
Windows Explorador de archivos y propiedades del recurso.
Linux Comandos 'smbd' y 'nmbd', configuración en /etc/samba/smb.conf.

Mini-Quiz

¿Cuál es el comando usado en Linux para compartir recursos mediante Samba?

  1. a) smbclient
  2. b) samba
  3. c) smbd
  4. d) nmbd

Checklist Configuración

  • Configurar el recurso a compartir.
  • Añadir permisos de acceso adecuados.
  • Probar la conexión desde otro equipo.
  • Verificar que los permisos están correctamente aplicados.

Actividad Práctica

Configurar una carpeta compartida en una red Linux y probar el acceso desde otro equipo.

Resumen

  • Recursos compartidos son archivos o dispositivos accesibles por varios usuarios.
  • Se pueden configurar en sistemas operativos libres y propietarios.
  • Buenas prácticas incluyen nombres descriptivos, permisos adecuados y actualizaciones.

Normativa General

Seguir las políticas de seguridad del lugar de trabajo al configurar recursos compartidos en red.
