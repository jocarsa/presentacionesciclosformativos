¿Qué es scripting?

Scripting es el uso de un lenguaje de programación para automatizar tareas repetitivas o complejas.

Lenguajes de scripting populares

  • Bash (Unix/Linux)
  • PowerShell (Windows)
  • Ruby
  • Perl

Cómo crear un script básico

  1. Elegir el lenguaje de scripting adecuado.
  2. Escribir el código en un editor de texto.
  3. Guardar el archivo con la extensión correspondiente (por ejemplo, . para).
  4. Hacer que el script sea ejecutable: chmod +x nombre_script.sh

Ejemplo de script en

#!/bin/bash
echo "Hola, mundo!"
date

Depuración de scripts

La depuración es el proceso de identificar y corregir errores en un script.

Métodos de depuración

  • Ejecución paso a paso (debugging)
  • Uso de mensajes de error personalizados
  • Pruebas unitarias
  • Revisión de código

Ejemplo de depuración en

#!/bin/bash
set -x  # Habilita el modo de depuración
echo "Hola, mundo!"
date
set +x  # Deshabilita el modo de depuración

Buenas prácticas en scripting

  • Comentar el código
  • Usar variables y funciones
  • Manejo de errores
  • Seguridad (evitar inyecciones)
  • Mantenimiento y actualización

Errores comunes en scripting

  • Sintaxis incorrecta
  • Comandos no reconocidos
  • Variables sin definir
  • Permisos insuficientes
  • Bucle infinito

Ejemplo de error común en

#!/bin/bash
echo "Hola, mundo!"
date
ls /ruta/incorrecta  # Error: directorio no encontrado

Comparativa entre sistemas operativos libres y propietarios

  • Sistemas libres (Linux): mayor personalización, seguridad, transparencia.
  • Sistemas propietarios (Windows, macOS): soporte técnico, facilidad de uso.

Scripting en sistemas operativos libres

Ejemplos:,, Ruby. Ventajas: mayor control, seguridad, comunidad activa.

Scripting en sistemas operativos propietarios

Ejemplos: PowerShell (Windows), AppleScript (macOS). Ventajas: facilidad de uso, soporte técnico.

Caso práctico: Automatización de tareas en Linux

Crear un script que actualice automáticamente los paquetes del sistema operativo.

Ejemplo de script para actualizar paquetes (Linux)

#!/bin/bash
sudo apt update && sudo apt upgrade -y

Caso práctico: Automatización de tareas en Windows

Crear un script que reinicie automáticamente el servicio de impresión.

Ejemplo de script para reiniciar servicio (Windows)

@echo off
net stop spooler
timeout /t 5
net start spooler
