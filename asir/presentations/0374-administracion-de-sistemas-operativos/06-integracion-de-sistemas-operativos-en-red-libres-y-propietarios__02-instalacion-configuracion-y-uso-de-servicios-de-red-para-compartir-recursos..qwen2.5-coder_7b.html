Portada

Integración de sistemas operativos en red libres y propietarios

Objetivo: Aprender a instalar, configurar y usar servicios de red para compartir recursos.

Instalación de servicios de red

  • Elegir el sistema operativo adecuado (libre o propietario)
  • Instalar el servidor de archivos (Samba, NFS, FTP)
  • Configurar las reglas de firewall
  • Instalar y configurar el servidor web (Apache, Nginx)
  • Instalar y configurar el servidor de correo electrónico (Postfix, Sendmail)

Ejemplo: Instalación de Samba en Linux


sudo apt-get update
sudo apt-get install samba
sudo smbpasswd -a username
sudo nano /etc/samba/smb.conf
# Añadir configuración del recurso compartido
sudo systemctl restart smbd nmbd

Configuración de servicios de red

  • Definir permisos de acceso (lectura, escritura)
  • Configurar grupos y usuarios para control de acceso
  • Ajustar la configuración del firewall para permitir el tráfico necesario
  • Optimizar la configuración según las necesidades específicas

Caso: Compartir carpetas en Windows y Linux

Configurar una carpeta compartida entre un sistema operativo Windows y uno basado en Linux usando Samba.

Proceso de configuración

  1. Instalar Samba en el servidor Linux
  2. Configurar los permisos de la carpeta compartida
  3. Añadir la carpeta al archivo smb.conf
  4. Reiniciar el servicio Samba
  5. Acceder a la carpeta desde un cliente Windows

Buenas prácticas de configuración

  • Usar contraseñas fuertes para los usuarios Samba
  • Limitar el acceso a grupos específicos
  • Monitorear y mantener actualizado el sistema
  • Documentar la configuración para futuras referencias

Errores comunes en la configuración

  • No definir permisos adecuados
  • No reiniciar el servicio Samba después de cambios
  • No configurar correctamente las reglas del firewall
  • Usar contraseñas débiles o fáciles de adivinar

Comparativa: Samba vs. NFS

Samba NFS
Protocolo de red basado en SMB/CIFS Protocolo de red basado en RPC
Soporta sistemas operativos Windows, Linux y macOS Principalmente para sistemas Unix/Linux
Más fácil de configurar para compartir archivos Más eficiente en términos de rendimiento
