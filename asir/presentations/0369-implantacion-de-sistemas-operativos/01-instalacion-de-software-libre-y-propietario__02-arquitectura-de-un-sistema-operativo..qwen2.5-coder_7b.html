Arquitectura de un Sistema Operativo

Objetivo: Comprender la estructura y componentes básicos de un sistema operativo.

Definición del Sistema Operativo

El Sistema Operativo (SO) es el software fundamental que gestiona los recursos del hardware y proporciona una interfaz entre el usuario y el hardware.

Estructura Básica de un SO

  • Kernel: Controlador central que administra los recursos del sistema.
  • Shell: Interfaz de línea de comandos para interactuar con el kernel.
  • Servicios y Daemons: Procesos en segundo plano que realizan tareas específicas.

Componentes del Kernel

  • Administración de memoria
  • Gestión de procesos
  • Control de dispositivos
  • Interfaz de red

Shell y Interpretes de Comandos

Los shells como o Z interpretan los comandos del usuario y ejecutan programas.

Servicios y Daemons

  • Apache: Servidor web
  • Mysql: Servidor de bases de datos
  • Cron: Programador de tareas en segundo plano

Ejemplo Práctico

Instalación del servidor web Apache en Ubuntu:

sudo apt update
sudo apt install apache2

Caso de Uso: Configuración de Red

Configurar una interfaz de red en Linux:

ip addr add 192.168.1.100/24 dev eth0
ip link set eth0 up

Proceso de Instalación de SO

  1. Selección del SO
  2. Preparación del hardware
  3. Descarga del instalador
  4. Ejecución del instalador
  5. Configuración inicial

Buenas Prácticas en Instalación

  • Usar discos separados para sistemas y datos
  • Copiar configuraciones importantes antes de la instalación
  • Verificar actualizaciones del SO después de la instalación

Errores Comunes en Instalación

  • No asignar suficiente espacio a los sistemas críticos
  • No configurar correctamente el firewall
  • No instalar actualizaciones del SO y software

Checklist: Instalación de SO

  1. Verificar compatibilidad del hardware con el SO
  2. Descargar la versión correcta del instalador
  3. Ejecutar el instalador en modo seguro (boot from USB/CD)
  4. Seleccionar opciones de partición cuidadosamente
  5. Realizar una copia de seguridad antes de la instalación
