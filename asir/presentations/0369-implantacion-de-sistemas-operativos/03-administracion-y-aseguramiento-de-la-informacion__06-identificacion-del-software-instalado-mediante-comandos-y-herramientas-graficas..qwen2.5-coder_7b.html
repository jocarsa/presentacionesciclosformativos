Portada

Identificación del software instalado mediante comandos y herramientas gráficas

Objetivo: Aprender métodos para identificar el software instalado en sistemas operativos.

Concepto de Identificación de Software

El proceso de identificación del software instalado en un sistema operativo permite determinar qué programas y aplicaciones están presentes en el equipo.

Herramientas Gráficas para la Identificación de Software

  • Panel de Control (Windows)
  • Aplicaciones (macOS)
  • Software Center o Gestor de Paquetes (Linux)

Herramientas Comandos para la Identificación de Software

  • Windows: `wmic product get name, version`
  • macOS: `pkgutil --pkgs`
  • Linux (Debian/Ubuntu): `dpkg -l`
  • Linux (Red Hat/CentOS): `rpm -qa`

Ejemplo de Identificación de Software en Windows

Usando el comando `wmic product get name, version`, se puede obtener una lista detallada de todos los productos instalados.

Ejemplo de Identificación de Software en macOS

El comando `pkgutil --pkgs` muestra todos los paquetes instalados en el sistema macOS.

Ejemplo de Identificación de Software en Linux (Debian/Ubuntu)

El comando `dpkg -l` lista todos los paquetes instalados en sistemas basados en Debian o Ubuntu.

Ejemplo de Identificación de Software en Linux (Red Hat/CentOS)

El comando `rpm -qa` muestra todos los paquetes RPM instalados en sistemas basados en Red Hat o CentOS.

Caso Práctico: Identificación de Software en un Sistema Operativo macOS

Caso Práctico: Identificación de Software en un Sistema Operativo Linux (Ubuntu)

Buenas Prácticas para la Identificación de Software

  • Realizar la identificación regularmente para mantener el control del software instalado.
  • Verificar la versión de los programas para asegurar actualizaciones y parches.
  • Eliminar software no utilizado para liberar espacio en disco.

Errores Comunes al Identificar Software

  • No tener permisos suficientes para ver todos los programas instalados.
  • Confundir la identificación del software con la lista de aplicaciones en uso.
  • No verificar las versiones de los programas instalados.

Mini-Quiz: Identificación de Software

¿Cuál comando se usa en Linux para listar todos los paquetes RPM instalados?

Checklist para la Identificación de Software

  • Identificar el sistema operativo en uso.
  • Elegir la herramienta gráfica o comando adecuada.
  • Ejecutar el comando o abrir la herramienta.
  • Verificar los resultados y asegurarse de que todos los programas están listados.
