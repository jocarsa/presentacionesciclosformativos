Instalación de sistemas operativos

Objetivo: Comprender los requisitos, versiones y licencias para la instalación de sistemas operativos.

Requisitos básicos para instalar un sistema operativo

  • Un ordenador con hardware compatible
  • Una conexión a internet (opcional, pero recomendada)
  • Un medio de almacenamiento (USB, DVD, CD)
  • Al menos 4 GB de espacio en disco duro

Versión del sistema operativo

Windows:

  • Windows 10 (64 bits)
  • Windows 11 (64 bits)

Linux:

  • Ubuntu
  • Debian
  • Fedora

Licencias de los sistemas operativos

Sistemas Operativos Propietarios:

  • Windows: Licencia de Microsoft
  • macOS: Licencia de Apple

Sistemas Operativos Libres y de Código Abierto:

  • Linux: Licencias como GPL, MIT, BSD
  • FreeBSD: Licencia 2-clause BSD

Ejemplo de instalación de Windows 10

  1. Descarga el ISO del sitio oficial de Microsoft
  2. Crea un USB bootable con el ISO
  3. Inicia el ordenador desde el USB
  4. Sigue las instrucciones en pantalla para la instalación

Ejemplo de instalación de Ubuntu

  1. Descarga el ISO del sitio oficial de Ubuntu
  2. Crea un USB bootable con el ISO
  3. Inicia el ordenador desde el USB
  4. Sigue las instrucciones en pantalla para la instalación

Buenas prácticas para la instalación

  • Realiza una copia de seguridad de los datos importantes antes de instalar
  • Verifica la integridad del ISO descargado
  • Elije un idioma y configuración regional adecuados

Errores comunes durante la instalación de sistemas operativos

  • Instalar el sistema operativo en el mismo disco que los datos
  • No asignar suficiente espacio para el sistema operativo
  • No configurar correctamente las particiones del disco

Comparativa entre sistemas operativos propietarios y libres

Sistemas Operativos Propietarios Sistemas Operativos Libres
Costo Pago anual por licencia Gratuito o con costos limitados
Personalización Menor personalización debido a restricciones de derechos de autor Más personalización y control sobre el código fuente

Mini-quiz: ¿Cuál es la ventaja principal de los sistemas operativos libres?

  1. Menor costo
  2. Más personalización y control sobre el código fuente
  3. Mejores actualizaciones y soporte técnico

Checklist para la instalación de un sistema operativo

  • Verifica los requisitos del sistema
  • Descarga el ISO correcto
  • Crea un USB bootable
  • Inicia el ordenador desde el USB
  • Sigue las instrucciones de instalación
