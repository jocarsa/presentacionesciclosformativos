Implantación de sistemas operativos

Objetivo: Comprender los equipos del dominio y su papel en la administración del acceso al dominio.

Definición de equipos del dominio

  • Servidores de dominio: Controlan el registro de nombres de usuario y grupos.
  • Computadoras de trabajo: Se unen al dominio para compartir recursos.
  • Servicios de red: Ofrecen funciones como impresión, almacenamiento compartido, etc.

Tipos de equipos del dominio

  • Equipo principal (Domain Controller): Almacena la base de datos del dominio.
  • Equipo secundario (Backup Domain Controller): Proporciona redundancia y respaldo.
  • Computadoras de trabajo: Se unen al dominio para compartir recursos.

Funciones del equipo principal (DC)

  • Gestión de cuentas de usuario y grupos.
  • Control de acceso a los recursos.
  • Sincronización de la hora.
  • Almacenamiento de información del dominio.

Funciones del equipo secundario (BDC)

  • Replícacion de la base de datos del dominio.
  • Sincronización de la hora.
  • Control de acceso a los recursos.
  • Gestión de cuentas de usuario y grupos.

Configuración del equipo principal (DC)

  1. Elegir el sistema operativo adecuado (Windows Server).
  2. Instalar el rol de Controlador de Dominio.
  3. Configurar la red y los nombres de dominio.
  4. Crear las unidades organizativas necesarias.

Configuración del equipo secundario (BDC)

  1. Elegir el sistema operativo adecuado (Windows Server).
  2. Instalar el rol de Controlador de Dominio.
  3. Configurar la red y los nombres de dominio.
  4. Sincronizar con el equipo principal.

Unión de computadoras al dominio

  1. Iniciar sesión en la computadora.
  2. Elegir la opción "Unirse a un dominio o red workgroup".
  3. Introducir el nombre del dominio y las credenciales de administrador.
  4. Reiniciar la computadora si es necesario.

Buenas prácticas en la configuración del dominio

  • Usar nombres de dominio descriptivos y fáciles de recordar.
  • Seguir políticas de seguridad para las cuentas de usuario.
  • Mantener actualizados los sistemas operativos y software.
  • Realizar copias de seguridad periódicas del dominio.

Errores comunes en la configuración del dominio

  • No usar nombres de dominio descriptivos.
  • No seguir políticas de seguridad para las cuentas de usuario.
  • No mantener actualizados los sistemas operativos y software.
  • No realizar copias de seguridad periódicas del dominio.

Comparativa entre equipos del dominio

Tipo de Equipo Papel en el Dominio
Equipo Principal (DC) Gestión central del dominio, almacenamiento de información.
Equipo Secundario (BDC) Replícacion de la base de datos, control de acceso.
Computadoras de Trabajo Unión al dominio, compartir recursos.
