Configuración y administración básica de «routers»

Objetivo: Aprender los comandos básicos para la administración de un router.

Introducción a los routers

  • Definición y función principal
  • Estructura básica de un router
  • Tipos de routers (domésticos, empresariales)

Conexiones básicas

  • Conexión Ethernet
  • Conexión Wi-Fi
  • Configuración de puertos

Configuración inicial

Proceso básico para configurar un router:

  1. Conectar el router a la red eléctrica
  2. Acceder a la interfaz web del router
  3. Cambiar la contraseña de administración

Configuración de puertos

  • Asignación de direcciones IP
  • Configuración de VLANs (opcional)
  • Configuración de puertos SSH y Telnet

Configuración de enrutamiento estático

Ejemplo básico:

        ip route 192.168.2.0 255.255.255.0 192.168.1.2

Configuración de enrutamiento dinámico

  • Protocolos de enrutamiento (OSPF, BGP)
  • Ejemplo básico de OSPF:
        router ospf 1
        network 192.168.1.0 0.0.0.255 area 0

Configuración de firewall básico

  • Definición y función del firewall
  • Ejemplo básico de reglas de firewall:
        access-list 1 permit 192.168.1.0 0.0.0.255
        interface GigabitEthernet0/0
        ip access-group 1 in

Configuración de DNS

  • Definición y función del DNS
  • Ejemplo básico de configuración:
        ip name-server 8.8.8.8
        ip domain-name example.com

Configuración de SSH y Telnet

Buenas prácticas:

  • Desactivar Telnet (seguridad)
  • Habilitar SSH (ifrado)
  • Configurar claves públicas/privadas

Errores comunes en la configuración de routers

  • Conectar el router a la red eléctrica incorrectamente
  • No cambiar la contraseña de administración
  • Configurar puertos incorrectamente

Buenas prácticas en la administración de routers

  • Mantener actualizaciones del firmware
  • Realizar copias de seguridad periódicas
  • Monitorear el rendimiento y los logs

Caso práctico: Configuración avanzada de un router

Ejemplo de configuración avanzada:

        interface GigabitEthernet0/1
        ip address 192.168.1.1 255.255.255.0
        no shutdown
        ip nat inside
        crypto isakmp policy 10
        encryption aes 256

Checklist de configuración básica de un router

  1. Conectar el router a la red eléctrica
  2. Acceder a la interfaz web del router
  3. Cambiar la contraseña de administración
  4. Configurar puertos Ethernet
  5. Configurar dirección IP y máscara de subred
