¿Qué es Ethernet?

Ethernet es una serie de tecnologías de red que utilizan el protocolo IEEE 802.3 para transmitir datos a través de cables.

Historia de Ethernet

  • Desarrollado por Xerox en los años 70
  • Estándar IEEE 802.3 publicado en 1985
  • Variaciones como Ethernet 10Base-T, 100Base-TX y 1000Base-T

Tipos de Ethernet

  • Ethernet 10Base-T: Velocidad de 10 Mbps, conexión a través de cables de cobre
  • Ethernet 100Base-TX: Velocidad de 100 Mbps, cableado cruzado
  • Ethernet 1000Base-T (Gigabit Ethernet): Velocidad de 1 Gbps

Características principales de Ethernet

  • Protocolo IEEE 802.3
  • Cableado cruzado (Cat5, Cat6)
  • Multipunto a multipunto
  • Duplex simplex y dúplex full

Ejemplo de configuración Ethernet

Configuración básica en un switch:

interface GigabitEthernet0/1
     description Link to Server
     ip address 192.168.1.1 255.255.255.0

Caso práctico: Implementación de Ethernet en una pequeña empresa

Se implementa Ethernet para conectar 10 computadoras a un servidor.

Proceso de instalación Ethernet

  1. Selección del tipo de cableado (Cat5, Cat6)
  2. Instalación de los cables
  3. Configuración de las interfaces en el switch y dispositivos
  4. Prueba de la conexión

Buenas prácticas Ethernet

  • Usar cableado cruzado correcto
  • Instalar cables en rutas adecuadas
  • Mantener la organización de los cables
  • Realizar pruebas periódicas

Errores comunes en Ethernet

  • Cableado cruzado incorrecto
  • Fallos de conexión física
  • Problemas de configuración IP
  • Sobrecarga del switch
