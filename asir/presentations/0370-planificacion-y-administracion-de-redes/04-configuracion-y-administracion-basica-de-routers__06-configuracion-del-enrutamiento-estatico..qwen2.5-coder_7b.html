Configuración del enrutamiento estático

Objetivo: Aprender los conceptos básicos y procedimientos para configurar el enrutamiento estático en routers.

Enrutamiento Estático vs Dinámico

  • Enrutamiento Estático: Configurado manualmente por un administrador.
  • Enrutamiento Dinámico: Se aprende automáticamente a través de protocolos como OSPF, BGP, etc.

Ventajas del Enrutamiento Estático

  • Predicción exacta de la ruta.
  • Reducción de carga en el router.
  • Control total sobre las rutas.

Desventajas del Enrutamiento Estático

  • No se actualiza automáticamente.
  • Puede requerir intervención manual si ocurren cambios en la red.
  • No es escalable para redes grandes.

Configuración Básica de Enrutamiento Estático

Se configura utilizando comandos como:

  • ip route [red] [máscara] [gateway]
  • ip route [red] [máscara] [next-hop]

Ejemplo de Configuración Estática

Router(config)# ip route 192.168.2.0 255.255.255.0 192.168.1.1
Router(config)# ip route 172.16.0.0 255.255.0.0 10.0.0.1

Verificación del Enrutamiento Estático

Se puede verificar con los siguientes comandos:

  • show ip route static
  • show ip route

Caso Práctico: Configuración de Enrutamiento Estático en una Red Pequeña

Configurar el enrutamiento estático en tres routers para conectarse entre sí y a una red remota.

Proceso de Configuración

  1. Determinar las redes y puertos necesarios.
  2. Configurar la dirección IP en cada router.
  3. Definir las rutas estáticas entre los routers.
  4. Verificar la configuración con comandos como show ip route.

Buenas Prácticas en Enrutamiento Estático

  • Usar prefijos de red correctos.
  • Configurar rutas alternativas para redundancia.
  • Mantener un registro de las configuraciones.

Errores Comunes en Enrutamiento Estático

  • Configurar rutas incorrectas o redundantes.
  • No definir rutas para todas las redes.
  • No tener configurada la ruta predeterminada.

Resumen del Enrutamiento Estático

  • Enrutamiento estático es manual y predeterminado.
  • Ventajas: control total, reducción de carga.
  • Desventajas: no se actualiza automáticamente, no escalable.
  • Se configura con comandos como ip route.

Normativa y Seguridad en Enrutamiento Estático

Seguir las normativas de seguridad al configurar el enrutamiento estático, incluyendo:

  • Uso de contraseñas fuertes.
  • Configuración de access-lists para controlar el tráfico.
  • Mantenimiento regular de las configuraciones.
