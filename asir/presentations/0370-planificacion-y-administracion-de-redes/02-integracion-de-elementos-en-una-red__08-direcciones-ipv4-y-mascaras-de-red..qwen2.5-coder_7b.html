Direcciones IPv4 y máscaras de red

Las direcciones IPv4 son identificadores únicos para dispositivos en una red.ada dirección IP consta de 32 bits, generalmente representados en notación decimal.

Estructura de una dirección IPv4

  • Formato: AAA.BBB.CCC.DDD (donde A, B, y D son números entre 0 y 255)
  • Cada octeto representa un grupo de 8 bits
  • Ejemplo: 192.168.1.1

Máscaras de red

Las máscaras de red se utilizan para determinar la parte de la dirección IP que identifica a la red y la parte que identifica al host.

Estructura de una máscara de red

  • Formato: AAA.BBB.CCC.DDD (donde A, B, y D son números entre 0 y 255)
  • Cada octeto puede tener valores como 255.255.255.0 o 255.0.0.0
  • Ejemplo: 255.255.255.0

Clases de direcciones IPv4

Las direcciones IPv4 se clasifican en A, B y según su rango de direcciones.

Clase A

  • Rango: 1.0.0.0 a 126.255.255.255
  • Máscara de red común: 255.0.0.0
  • Usada para redes grandes

Clase B

  • Rango: 128.0.0.0 a 191.255.255.255
  • Máscara de red común: 255.255.0.0
  • Usada para redes medias

Clase

  • Rango: 192.0.0.0 a 223.255.255.255
  • Máscara de red común: 255.255.255.0
  • Usada para redes pequeñas

Direcciones especiales

  • 127.0.0.0: Dirección de bucle local (localhost)
  • 169.254.0.0/16: Direcciones autoconfiguradas
  • 224.0.0.0/4: Multicast

Cálculo de subredes

Elálculo de subredes permite dividir una red en segmentos más pequeños, mejorando la gestión y el rendimiento.

Ejemplo deálculo de subred

  • Dirección IP: 192.168.1.0/24
  • Máscara de red: 255.255.255.0
  • Subredes posibles: 254 (menos 2 para la red y el broadcast)

Buenas prácticas en asignación de direcciones IP

  • Evitar direcciones de clase D y E
  • Utilizar máscaras de red adecuadas para el tamaño de la red
  • Asignar direcciones reservadas a redes privadas

Errores comunes en asignación de direcciones IP

  • Confusión entre clases A, B y
  • Uso incorrecto de máscaras de red
  • No asignación de direcciones reservadas a redes privadas

Caso práctico: Asignación de direcciones IP en una pequeña empresa

Una pequeña empresa con 50 usuarios puede usar una red de clase (192.168.1.0/24) y asignar direcciones IP desde 192.168.1.1 hasta 192.168.1.50.

Resumen: Direcciones IPv4 y máscaras de red

  • Direcciones IPv4 son identificadores únicos para dispositivos en una red.
  • Máscaras de red determinan la parte de la dirección que identifica a la red y al host.
  • Existen clases A, B y con máscaras de red específicas.

Normativa: Asignación de direcciones IP

La asignación de direcciones IP debe seguir las normativas estableidas por la IETF y los organismos de regulación correspondientes.
