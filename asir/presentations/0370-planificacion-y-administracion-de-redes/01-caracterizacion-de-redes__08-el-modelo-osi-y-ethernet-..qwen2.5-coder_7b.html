Modelo OSI (Open Systems Interconnection)

  • Divide la comunicación en 7 capas.
  • Cada capa tiene una función específica.
  • Ejemplos de capas: Aplicación, Presentación, Sesión, Transporte, Red, Enlace de datos y Física.

Capa 7: Aplicación

Se ocupa del protocolo de aplicación (como HTTP o FTP).

Interactúa directamente con los usuarios finales.

Capa 6: Presentación

Se encarga de la codificación y descodificación de datos.

Maneja formatos como JPEG, MP3, etc.

Capa 5: Sesión

Gestiona las conexiones entre aplicaciones.

Mantiene sesiones de trabajo activas.

Capa 4: Transporte

Se encarga de la transmisión confiable de datos.

Gestiona errores y retransmisiones.

Capa 3: Red

Maneja el enrutamiento de paquetes entre redes.

Distribuye datos según la dirección IP.

Capa 2: Enlace de datos

Gestiona la transmisión de bits a través del medio físico.

Controla errores en el nivel de bit.

Capa 1: Física

Se ocupa del medio físico y la transmisión de señales eléctricas o ópticas.

Ejemplos: Ethernet, Wi-Fi, Fibra óptica.

Ethernet

Protocolo de red de nivel 2 del modelo OSI.

Se utiliza en redes locales (LANs).

Ejemplos de Ethernet

  • Ethernet 10Base-T
  • Ethernet 100Base-TX
  • Ethernet 1000Base-SX

Características de Ethernet

  • Rendimiento alto (1 Gbps)
  • Capacidad para grandes volúmenes de datos
  • Soporta redes de hasta 100 metros
