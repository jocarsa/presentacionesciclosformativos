Dirección de redes

La dirección de redes es el proceso de asignar direcciones IP a los dispositivos en una red para permitir la comunicación entre ellos.

Tipos de direcciones IP

  • Dirección IPv4: 32 bits (4 octetos)
  • Dirección IPv6: 128 bits (8 grupos de 4 hexadecimales)

Clases de direcciones IP

  • A: 0.0.0.0 - 127.255.255.255 (1 bit para la clase)
  • B: 128.0.0.0 - 191.255.255.255 (2 bits para la clase)
  • C: 192.0.0.0 - 223.255.255.255 (3 bits para la clase)

Subredes y máscaras de subred

Las máscaras de subred permiten dividir una red en subredes más pequeñas.

Cálculo de la máscara de subred

  • Determinar el número de bits de red (clase)
  • Calcular el número de bits de host
  • Convertir los bits a notación CIDR

Ejemplo deálculo de máscara

Dado una red A (24 bits de red), la máscara es 255.255.255.0.

Uso de direcciones especiales

  • Dirección de broadcast (último bit del bloque de host en 1)
  • Dirección de red (primer bit del bloque de host en 0)
  • Direcciones reservadas para otros usos

Configuración de direcciones IP

Puede realizarse manualmente o automáticamente mediante DHCP.

Ejemplo de configuración manual

  • Dirección IP: 192.168.1.100
  • Máscara de subred: 255.255.255.0
  • Gateway: 192.168.1.1
  • DNS: 8.8.8.8, 8.8.4.4

Ejemplo de configuración DHCP

El dispositivo obtiene automáticamente la dirección IP y otros parámetros del servidor DHCP.

Comandos útiles en Windows

  • ipconfig /all
  • ping [dirección IP]
  • tracert [dirección IP]

Comandos útiles en Linux

  • ifconfig
  • ping [dirección IP]
  • traceroute [dirección IP]

Buenas prácticas de dirección de redes

  • Usar subredes lo más pequeñas posible para optimizar la eficiencia
  • Asegurarse de que las direcciones IP no se repitan en diferentes subredes
  • Reservar direcciones para servidores y dispositivos críticos

Errores comunes en la dirección de redes

  • Dirección IP incorrecta o no asignada
  • Máscara de subred incompatibles
  • No configuración de gateway

Mini-quiz: Dirección de redes

¿Cuál es la máscara de subred para una red con 254 dispositivos?

  1. 255.255.255.0
  2. 255.255.255.128
  3. 255.255.255.240

Checklist: Dirección de redes

  • Dirección IP correcta asignada a cada dispositivo
  • Máscara de subred adecuada para la cantidad de dispositivos
  • Configuración de gateway y DNS correctas
  • No repeticiones de direcciones IP en diferentes subredes

Resumen

  • Dirección IP: Identifica cada dispositivo en la red
  • Máscaras de subred: Divide la red en segmentos más pequeños
  • Configuración correcta es crucial para la comunicación entre dispositivos

Normativa

La normativa ISO/IE 25964 establece las recomendaciones para la dirección de redes.
