Configuración y administración básica de «routers»

Objetivo: Comprender los roles y funciones de los routers en redes LAN y WAN.

¿Qué son los Routers?

Los routers son dispositivos electrónicos que se utilizan para interconectar diferentes segmentos de una red, permitiendo la comunicación entre ellas.

Funciones principales de un Router

  • Enrutamiento: Determina el camino más eficiente para los paquetes de datos.
  • Filtración de tráfico: Controla qué datos pueden pasar entre redes.
  • Conmutación de paquetes: Envía paquetes individuales a través de la red.

Tipos de Routers

  • Routers domésticos (residenciales): Para hogares y pequeñas oficinas.
  • Routers empresariales: Para redes corporativas más grandes.
  • Routers de cortocircuito: Para interconectar diferentes partes de una misma red.

Configuración Básica de un Router

Incluye la configuración de direcciones IP, puertos, protocolos y seguridad.

Ejemplo: Configurando una Dirección IP en un Router

ip address 192.168.1.1 255.255.255.0

Caso Práctico: Configuración de un Router para una LAN

Configurar un router para conectar varias computadoras en la misma red local.

Proceso de Enrutamiento

  • Determina el camino más corto a través de las tablas de enrutamiento.
  • Usa protocolos como OSPF, BGP o RIP para actualizar las tablas.

Buenas Prácticas al Administrar Routers

  • Mantén actualizadas las firmas del software y los parches de seguridad.
  • Configura contraseñas fuertes para el acceso administrativo.
  • Realiza copias de seguridad periódicas de la configuración.

Errores Comunes al Administrar Routers

  • No cambiar las contraseñas por defecto.
  • No realizar copias de seguridad regulares.
  • No mantener el software actualizado.

  1. Configurar dirección IP.
  2. Establecer puertos y protocolos.
  3. Aplicar seguridad (contraseñas, firewall).
  4. Realizar copias de seguridad.

Resumen

  • Los routers son dispositivos que enrutan paquetes de datos entre redes.
  • Se utilizan tanto en LAN como en WAN.
  • Buenas prácticas incluyen actualizaciones, contraseñas fuertes y copias de seguridad.
