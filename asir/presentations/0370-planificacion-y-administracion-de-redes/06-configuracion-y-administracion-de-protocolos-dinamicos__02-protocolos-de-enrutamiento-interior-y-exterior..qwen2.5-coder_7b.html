Configuración y administración de protocolos dinámicos

Objetivo: Comprender los protocolos de enrutamiento interior y exterior, su configuración y administración.

Protocolos de enrutamiento interior

  • RIP (Routing Information Protocol)
  • OSPF (Open Shortest Path First)
  • BGP (Border Gateway Protocol)
  • EIGRP (Enhanced Interior Gateway Routing Protocol)
  • IS-IS (Intermediate System to Intermediate System)

RIP: Protocolo de enrutamiento interior

RIP utiliza un algoritmo de vecinos para propagar información sobre rutas.

OSPF: Protocolo de enrutamiento interior

OSPF es un protocolo de enrutamiento de área que usa el algoritmo SPF (Shortest Path First).

BGP: Protocolo de enrutamiento exterior

BGP se utiliza para interconectar diferentes redes IP y es conocido por su complejidad.

EIGRP: Protocolo de enrutamiento interior

EIGRP es un protocolo de enrutamiento que combina características de OSPF y RIP.

IS-IS: Protocolo de enrutamiento interior

IS-IS es un protocolo de enrutamiento de nivel 2 que usa el algoritmo SPF.

Ejemplo de configuración RIP

router rip
 network 192.168.1.0
 version 2

Ejemplo de configuración OSPF

router ospf 1
 network 192.168.1.0 0.0.0.255 area 0

Ejemplo de configuración BGP

router bgp 65000
 neighbor 192.168.1.2 remote-as 65001

Ejemplo de configuración EIGRP

router eigrp 1
 network 192.168.1.0

Ejemplo de configuración IS-IS

router isis
 net 49.0001.0000.0000.0001.00

Buenas prácticas en la configuración de protocolos dinámicos

  • Configurar correctamente los parámetros de cada protocolo.
  • Monitorear el estado del enrutamiento regularmente.
  • Ajustar los valores de métrica según sea necesario.
  • Evitar la configuración de bucles en las rutas.

Errores comunes en la configuración de protocolos dinámicos

  • No especificar correctamente los rangos de red.
  • No configurar las interfaces correctas.
  • No establecer las métricas adecuadas.
  • Olvidar la configuración de vecinos o routers BGP.

Comparativa entre protocolos dinámicos

Protocolo Ventajas Inconvenientes
RIP Sencillo, rápido de configurar. Limitado en tamaño de red, no soporta VLSM.
OSPF Soporta VLSM, escalable. Más complejo que RIP, requiere más configuración.
BGP Escalable para redes grandes. Complejo de configurar y mantener.

  • Configurar los rangos de red correctamente.
