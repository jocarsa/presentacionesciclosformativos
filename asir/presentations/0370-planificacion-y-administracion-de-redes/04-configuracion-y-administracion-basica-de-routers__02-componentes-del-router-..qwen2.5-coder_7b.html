Configuración y administración básica de «routers»

Objetivo: Comprender los componentes básicos de un router y su configuración inicial.

Componentes del Router

  • CPU (Procesador): Controla todas las operaciones del router.
  • Memoria RAM: Almacena programas y datos temporales.
  • Memoria Flash/ROM: Almacena el sistema operativo y configuración permanente.
  • Puertos de Conexión: Ethernet, Wi-Fi, USB, etc., para conectar dispositivos.
  • Cableado Interno: Conecta los componentes internos del router.
  • Batería (si es modelo portátil): Proporciona energía cuando no está conectado a la corriente.

CPU y Memoria RAM

La CPU ejecuta los programas del sistema operativo, mientras que la memoria RAM almacena datos temporales para su acceso rápido.

Memoria Flash/ROM

Almacena el sistema operativo y configuración del router. La ROM es de solo lectura, mientras que la Flash puede ser modificada.

Puertos de Conexión

  • Ethernet: Para conectar dispositivos a través de cables Ethernet.
  • Wi-Fi: Para conectar dispositivos inalámbricamente.
  • USB: Para cargar dispositivos o transferir archivos.
  • Console Port: Para conexión directa a través de un cable serial.

Cableado Interno

Conecta los componentes internos del router, como la CPU, memoria y puertos de red.

Batería (Portátiles)

Proporciona energía cuando el router no está conectado a la corriente eléctrica.

Ejemplo de Configuración Inicial

Después de conectar el router, se accede al navegador web y se introduce la dirección IP por defecto para configurar el dispositivo.

Caso Práctico: Cambio de Contraseña

Para mejorar la seguridad, es recomendable cambiar la contraseña predeterminada del router.

Proceso de Configuración Básica

  1. Conectar el router a una fuente de alimentación.
  2. Acceder al navegador web y introducir la dirección IP del router.
  3. Ingresar las credenciales de inicio de sesión (generalmente "admin" para ambos).
  4. Modificar la configuración según sea necesario (red, seguridad, etc.).

Buenas Prácticas en Configuración

  • Cambiar la contraseña predeterminada.
  • Desactivar el acceso remoto si no es necesario.
  • Actualizar regularmente el firmware del router.

Errores Comunes en Configuración

  • No cambiar la contraseña predeterminada.
  • Habilitar el acceso remoto sin seguridad adecuada.
  • No mantener actualizado el firmware del router.

Checklist de Configuración Inicial

  1. Conectar el router a la corriente.
  2. Ingresar al navegador web y la dirección IP del router.
  3. Cambiar la contraseña predeterminada.
  4. Desactivar el acceso remoto si no es necesario.

Resumen

  • Los routers tienen varios componentes importantes: CPU, memoria RAM, memoria Flash/ROM, puertos de conexión y cableado interno.
  • La configuración inicial incluye acceso al router, cambio de contraseña y desactivar accesos no seguros.
  • Buenas prácticas incluyen seguridad, actualizaciones regulares y evitar errores comunes.

Normativa General

La configuración de redes debe seguir normativas locales y internacionales para garantizar la seguridad y el funcionamiento adecuado del sistema.
