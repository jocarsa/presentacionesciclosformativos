Formas de conexión al router para su configuración inicial

Objetivo: Aprender las diferentes formas de conectarse a un router para realizar su configuración inicial.

Conexión por consola (Console)

  • Usa un cable Ethernet o RS-232.
  • Permite una conexión directa y segura al router.
  • No requiere configuración de IP en el equipo cliente.
  • Se utiliza para configuraciones básicas y diagnóstico.

Conexión por SSH (Secure Shell)

  • Usa un cable Ethernet o Wi-Fi.
  • Permite una conexión segura a través de Internet.
  • Necesita configuración previa del router y equipo cliente.
  • Más flexible que la consola, pero requiere conocimientos técnicos.

Conexión por Telnet

  • Usa un cable Ethernet o Wi-Fi.
  • Permite una conexión insegura a través de Internet.
  • Necesita configuración previa del router y equipo cliente.
  • No recomendado para producción debido a la falta de encriptación.

Conexión por Web (Web Interface)

  • Accede mediante un navegador web.
  • No requiere conexión física al router.
  • Más accesible para usuarios no técnicos.
  • Puede requerir configuración previa de IP en el equipo cliente.

Conexión por USB (USB to Serial)

  • Usa un adaptador USB a serial.
  • Permite una conexión directa y segura al router.
  • No requiere configuración de IP en el equipo cliente.
  • Más costoso que la consola, pero ofrece más funcionalidades.

Conexión por Wi-Fi (Wireless)

  • Usa una conexión inalámbrica.
  • No requiere cableado físico.
  • Puede requerir configuración previa de IP en el equipo cliente.
  • Más accesible para usuarios no técnicos.

Conexión por SNMP (Simple Network Management Protocol)

  • Usa un protocolo de gestión de red.
  • No requiere conexión física al router.
  • Más complejo que la interfaz web.
  • Puede requerir configuración previa del router y equipo cliente.
