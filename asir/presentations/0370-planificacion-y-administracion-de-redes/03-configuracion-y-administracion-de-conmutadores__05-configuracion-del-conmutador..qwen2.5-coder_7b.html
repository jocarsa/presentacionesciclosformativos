Conectando el Conmutador a la Red

Pasos para conectar un conmutador a una red existente:

  1. Encender el conmutador
  2. Configurar la conexión Ethernet
  3. Asignar una dirección IP

Configuración Básica del Conmutador

Comandos básicos para configurar un conmutador:

  • Entrada en modo privilegiado: enable
  • Cambio de nombre del conmutador: hostname
  • Configuración de la contraseña: enable secret

Interfaz de Red del Conmutador

Concepto y tipos de interfaces:

  • Ethernet
  • FastEthernet
  • GigabitEthernet
  • Serial

Configuración de Interfaces del Conmutador

Pasos para configurar una interfaz:

  1. Entrada en modo de configuración de interfaz: interface
  2. Asignación de dirección IP
  3. Habilitación de la interfaz: no shutdown

Configuración de Puertos del Conmutador

Tipos de puertos y sus funciones:

  • Puerto Access
  • Puerto Trunk
  • Puerto Hybrid

Configuración de VLANs en el Conmutador

Pasos para configurar VLANs:

  1. Crear una VLAN: vlan
  2. Asignar puertos a una VLAN: switchport mode access/trunk/hybrid
  3. Configurar el nombre de la VLAN: name

Configuración de Protocolos de Enrutamiento en el Conmutador

Protocolos comunes y sus funciones:

  • STP (Spanning Tree Protocol)
  • RIP (Routing Information Protocol)
  • EIGRP (Enhanced Interior Gateway Routing Protocol)

Configuración de Seguridad en el Conmutador

Pasos para mejorar la seguridad del conmutador:

  1. Habilitar la autenticación SSH/Telnet
  2. Crear listas de acceso (ACL)
  3. Configurar el firewall

Verificación y Diagnóstico del Conmutador

Comandos para verificar el estado del conmutador:

  • show ip interface brief
  • show vlan
  • show spanning-tree

Buenas Prácticas en la Configuración del Conmutador

  • Documentar todas las configuraciones
  • Hacer copias de seguridad periódicas
  • Mantener actualizadas las firmas del software

Errores Comunes en la Configuración del Conmutador

  • No asignar una dirección IP a la interfaz
  • Habilitar el conmutador sin configurar las interfaces
  • No habilitar puertos necesarios

Mini-Quiz: Configuración del Conmutador

Pregunta: ¿Qué comando se utiliza para cambiar el nombre de un conmutador?

  1. hostname
  2. rename
  3. set name

Checklist: Configuración del Conmutador
