Objetivo: Aprender los comandos básicos para configurar un router.

Introducción a la Configuración de Routers

  • Importancia de la configuración correcta de routers en redes empresariales.
  • Concepto básico: qué es un router y su función.
  • Ejemplo práctico: cómo se conectan los dispositivos en una red con un router.

Configuración Básica de un Router

  1. Encender el router y acceder a su interfaz web.
  2. Cambiar la contraseña predeterminada del administrador.
  3. Configurar la dirección IP del router.
  4. Establecer la configuración de red (LAN/WAN).

Comandos Básicos en el Shell del Router

  • show ip interface brief: Muestra las interfaces IP activas.
  • ping [dirección_IP]: Prueba la conectividad con otra máquina.
  • traceroute [dirección_IP]: Rastrea el camino de datos a una dirección IP.

Configuración de Puertos y VLANs

  1. Configurar puertos en modo Access o Trunk.
  2. Crear y asignar VLANs (VLAN ID, nombre).
  3. Configurar interfaces para diferentes VLANs.

Configuración de Rutas Estáticas

  • ip route [red] [máscara] [puerta_de_enlace]: Configura una ruta estática.
  • Ejemplo: ip route 192.168.2.0 255.255.255.0 192.168.1.1.

  • BGP (Border Gateway Protocol):
  • Ejemplo: Configurar OSPF en un router.

Configuración de Firewall y Seguridad

  1. Habilitar el firewall del router.
  2. Configurar reglas de acceso (ACL).
  3. Ejemplo: Permitir solo tráfico SSH al router.

Backup y Restauración de Configuraciones

  • show running-config: Muestra la configuración actual en ejecución.
  • write memory o copy running-config startup-config: Guarda la configuración.
  • Ejemplo: Restaurar una configuración guardada.

Buenas Prácticas en la Configuración de Routers

  • Documentar todas las configuraciones realizadas.
  • Realizar copias de seguridad periódicas.
  • Monitorear el rendimiento del router.

Errores Comunes en la Configuración de Routers

  • Configurar interfaces incorrectamente (LAN/WAN).
  • No configurar correctamente las VLANs.
  • Olvidar el backup periódico de la configuración.
