Caracterización de redes

Objetivo: Comprender los tipos de redes, topologías y arquitecturas.

Definición de redes LAN, MAN y WAN

  • LAN (Local Area Network): Red pequeña que conecta dispositivos físicamente cercanos.
  • MAN (Metropolitan Area Network): Red que abarca una ciudad o área metropolitana.
  • WAN (Wide Area Network): Red que cubre grandes distancias, como países o continentes.

Ejemplos de redes LAN

  • Casa con computadoras y dispositivos conectados.
  • Oficina pequeña con ordenadores y impresoras.
  • Sala de conferencias en una empresa.

Ejemplos de redes MAN

  • Ciudad universitaria.
  • Barrio residencial grande.
  • Zona comercial en una ciudad.

Ejemplos de redes WAN

  • Servicio de internet a domicilio.
  • Conexión entre dos ciudades.
  • Comunicación global entre empresas.

Topologías de redes

  • Estrella: Un nodo central conectado a todos los demás nodos.
  • Ring: Dispositivos conectados en unírculo cerrado.
  • Torre (Mesh):ada dispositivo conectado a todos los demás.

Ejemplos de topología estrella

  • Red doméstica con un router central.
  • Sala de conferencias con un switch principal.

Ejemplos de topología ring

  • Cableado de fibra óptica en una oficina pequeña.
  • Red de computadoras en una escuela primaria.

Ejemplos de topología torre (mesh)

  • Red empresarial con múltiples switches interconectados.
  • Sistema de seguridad en una gran empresa.

Arquitecturas de redes

  • Cliente-Servidor: Un servidor central proporciona recursos a múltiples clientes.
  • Peer-to-Peer (P2P): Dispositivos interambian recursos directamente entre sí.
  • Red de Área Amplia (MAN): Red que abarca una ciudad o área metropolitana.

Ejemplos de arquitectura cliente-servidor

  • Sistema de correo electrónico corporativo.
  • Red de impresoras compartidas en una oficina.

Ejemplos de arquitectura P2P

  • Descarga de archivos usando BitTorrent.
  • Sistema de chat en tiempo real como WhatsApp.

Ejemplos de arquitectura MAN

  • Ciudad universitaria con red de computadoras.
  • Barrio residencial grande con sistema de seguridad.

Protocolos de redes

  • TCP/IP: Protocolo de internet que permite la comunicación global.
  • HTTP/HTTPS: Protocolos para el acceso a páginas web.
  • FTP: Protocolo para transferencia de archivos.

Ejemplos de protocolo TCP/IP

  • Comunicación entre dos computadoras en una LAN.
  • Acceso a internet desde un dispositivo móvil.

Ejemplos de protocolo HTTP/HTTPS

  • Navegación web en un navegador.
  • Descarga de archivos desde una página web.

Ejemplos de protocolo FTP

  • Carga y descarga de archivos entre dos computadoras en una LAN.
  • Transferencia de archivos a un servidor web.
