Clasificación de aplicaciones web

Objetivo: Comprender las principales categorías y tipos de aplicaciones web.

Categoría por funcionalidad

  • E-commerce
  • Social media
  • Portales informativos
  • Gestión empresarial
  • Aplicaciones móviles web

Categoría por tecnología utilizada

  • Front-end:, CSS,
  • Back-end:,, Ruby, Node.js
  • Bases de datos: SQL, NoSQL
  • Servidores: Apache, Nginx

Categoría por arquitectura

  • Monolítica
  • Microservicios
  • Serverless
  • Full-stack
  • Desarrollo ágil

Ejemplo de aplicación web e-commerce

Amazon.com: Tienda en línea con carrito de compras, sistema de pago y seguimiento del pedido.

Ejemplo de aplicación web social media

Facebook: Plataforma para compartir contenido, interactuar con amigos y familiares.

Ejemplo de aplicación web portal informativo

La Casa Rosada: Sitio oficial del gobierno que proporciona información sobre políticas públicas.

Ejemplo de aplicación web gestión empresarial

SAP Business One: Software para la gestión integral de negocios.

Ejemplo de aplicación web móvil web

Spotify: Servicio de música disponible en navegadores móviles.

Buenas prácticas en la clasificación de aplicaciones web

  • Consistencia en las categorías utilizadas
  • Actualización regular de la clasificación
  • Consideración del contexto y objetivo de cada aplicación

Errores comunes en la clasificación de aplicaciones web

  • No considerar las funcionalidades específicas de la aplicación
  • Mixing tecnologías y arquitecturas sin diferenciación clara
  • Ignorar el contexto en el que se utiliza la aplicación
