Creación de una Base de Datos en MySQL

Ejemplo de comando SQL:

CREATE DATABASE nombre_base_de_datos;

Selección de la Base de Datos

Ejemplo de comando SQL:

USE nombre_base_de_datos;

Creación de Tablas en MySQL

Ejemplo de comando SQL para crear una tabla:

CREATE TABLE nombre_tabla (
        id INT AUTO_INCREMENT PRIMARY KEY,
        nombre VARCHAR(100),
        edad INT
    );

Estructura de una Tabla

  • Campos (columnas)
  • Tipo de datos
  • Restriciones (llaves primarias, únicas, etc.)

Ejemplos de Tipos de Datos

  • INT
  • VARCHAR(n)
  • DATE
  • DECIMAL(p, s)

Añadir Campos a una Tabla Existente

Ejemplo de comando SQL:

ALTER TABLE nombre_tabla ADD COLUMN nuevo_campo INT;

Borrar Campos de una Tabla

Ejemplo de comando SQL:

ALTER TABLE nombre_tabla DROP COLUMN campo_a_eliminar;

Modificar Campos de una Tabla

Ejemplo de comando SQL:

ALTER TABLE nombre_tabla MODIFY COLUMN campo INT NOT NULL;

Insertar Datos en una Tabla

Ejemplo de comando SQL:

INSERT INTO nombre_tabla (nombre, edad) VALUES ('Juan', 30);

Consultar Datos en una Tabla

Ejemplo de comando SQL:

SELECT * FROM nombre_tabla WHERE edad > 25;

Borrar Datos de una Tabla

Ejemplo de comando SQL:

DELETE FROM nombre_tabla WHERE id = 1;

Actualizar Datos en una Tabla

Ejemplo de comando SQL:

UPDATE nombre_tabla SET edad = 31 WHERE id = 1;

Resumen

Buenas Prácticas

  • Usar nombres descriptivos para tablas y campos
  • Asegurar la seguridad de las bases de datos
  • Mantener un diseño normalizado

Errores Comunes

  • No cerrar correctamente las conexiones a la base de datos
  • No validar los datos antes de insertarlos
  • No manejar adecuadamente las excepciones

Mini-Quiz

¿Cuál es el comando SQL para crear una base de datos?

  1. CREATE TABLE nombre_base_de_datos;
  2. CREATE DATABASE nombre_base_de_datos;
  3. SELECT * FROM nombre_base_de_datos;
  4. UPDATE nombre_base_de_datos SET nombre = 'nombre_nuevo';

Checklist de Creación de Tablas

  • Especificar el nombre de la tabla
  • D definir los campos con sus tipos y restricciones
  • Asegurarse de que las llaves primarias sean únicas
  • Validar los datos antes de insertarlos

Actividad Práctica

Crea una base de datos llamada "biblioteca" y dos tablas: "libros" y "autores". La tabla "libros" debe tener campos como id, título, autor_id (llave foránea), año_publicacion. La tabla "autores" debe tener campos como id, nombre, nacionalidad.

Normativa y Seguridad

  • Cryptografía de datos sensibles
