Lenguajes de Marca y Script

Los lenguajes de marca (,) definen la estructura y el contenido de una página web. Los lenguajes de script (,) añaden funcionalidad dinámica.

Ejemplo Básico: en

<?
    $nombre = "Juan";
  ?>
  <p>Hola, <? echo $nombre; ?>!</p>

Buenas Prácticas

  • Mantener el código limpio y organizado.
  • Evitar la mezcla excesiva de lógica y presentación.
  • Utilizar variables y funciones para mejorar la reutilización del código.

Errores Comunes

  • Mixing y sin cerrar correctamente las etiquetas.
  • No actualizando el navegador después de cambios en el código.
  • No controlando errores en la lógica del script.
