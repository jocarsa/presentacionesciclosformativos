Objetivo: Comprender y configurar los mecanismos de seguridad integrados en aplicaciones web.

Concepto de seguridad web

La seguridad web se refiere a las medidas que se toman para proteger la información sensible almacenada y transmitida en una aplicación web.

Cifrado SSL/TLS

  • Protege la comunicación entre el cliente y el servidor.
  • Usa certificados digitales para autenticar al servidor.
  • Encuentra su configuración en los archivos de configuración del servidor web (Apache, Nginx).

Autenticación y autorización

La autenticación verifica la identidad del usuario. La autorización determina qué acciones puede realizar.

Cookies seguras

  • Las cookies deben ser marcadas como "Secure" y "HttpOnly".
  • Esto impide que las cookies sean accesibles a través de.
  • Configuración en el archivo de configuración del servidor web.

CAPTCHAs y medidas contra bots

CAPTCHAs son herramientas para prevenir accesos no autorizados. Otros métodos incluyen ratelimiting y análisis de comportamiento.
