Sintaxis en Programación de Documentos Web

Objetivo: Aprender las reglas básicas y sintaxis para escribir scripts de servidor.

¿Qué es la Sintaxis?

Sintaxis se refiere a las reglas gramaticales que determinan cómo se estructuran los elementos en un lenguaje de programación.

Ejemplo de Sintaxis en

<?
echo "Hola, mundo!";
?>

Reglas Básicas de Sintaxis

  • Cada instrucción termina con un punto y coma (;)
  • Las variables comienzan con el signo dolar ($)
  • No se distingue entre mayúsculas y minúsculas en las palabras clave

Ejemplo de Variables en

<?
$nombre = "Juan";
echo $nombre;
?>

Comentarios en

  • Comentarios de una línea: // texto
  • Comentarios multilínea: /* texto */

Ejemplo de Comentarios en

<?
// Esto es un comentario de una línea
/* 
Esto es un 
comentario multilínea
*/
echo "Hola, mundo!";
?>

Operadores en

  • Aritméticos: +, -, *, /, %
  • Comparación: ==, !=, >, <, >=, <=
  • Lógicos: &&, ||, !

Ejemplo de Operadores en

<?
$a = 5;
$b = 10;
echo $a + $b; // Suma
echo $a > $b; // Comparación
echo !$a; // Negación lógica
?>

Funciones en

Las funciones son bloques de código reutilizables que realizan una tarea específica.

Ejemplo de Función en

<?
function saludo($nombre) {
    return "Hola, $nombre!";
}
echo saludo("Juan");
?>

Control de Flujo en

  • if-else: condicionales
  • for: bucles iterativos
  • foreach: bucles para arrays

Ejemplo de Control de Flujo en

<?
$numero = 5;
if ($numero > 0) {
    echo "El número es positivo";
} else {
    echo "El número es negativo";
}
?>

Errores Comunes en Sintaxis

  • Falta de punto y coma (;)
  • Error de sintaxis en las variables
  • Comentarios mal formados

Ejemplo de Errores Comunes

<?
echo "Hola, mundo!" // Falta punto y coma
?>

Buenas Prácticas en Sintaxis

  • Usar espacios entre operadores y variables
  • Comentar el código para facilitar su lectura
  • Mantener un orden lógico en el código

Ejemplo de Buenas Prácticas

<?
// Comentar el código para facilitar su lectura
$numero = 5; // Declaración de variable

if ($numero > 0) { // Condicional
    echo "El número es positivo"; // Salida
} else {
    echo "El número es negativo";
}
?>

Checklist: Sintaxis en

  • [ ] Usar punto y coma al final de cada instrucción
  • [ ] Declarar variables con el signo dolar ($)
  • [ ] Comentar el código para facilitar su lectura
  • [ ] Mantener un orden lógico en el código

Resumen

  • utiliza variables, operadores y funciones para realizar tareas.
