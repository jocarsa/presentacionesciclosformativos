Instalación y Configuración de un Sistema Gestor de Base de Datos en Servidores Web

Objetivo: Aprender los pasos para instalar y configurar un sistema gestor de base de datos (SGBD) en servidores web.

Introducción a los Sistemas Gestores de Base de Datos (SGBD)

  • SGBD son herramientas que permiten almacenar, gestionar y recuperar datos de manera eficiente.
  • Ejemplos populares: MySQL, PostgreSQL, Oracle Database, Microsoft SQL Server.
  • Importancia en la arquitectura web para almacenar información del usuario, configuraciones, etc.

Requisitos Previos

  • Servidor web instalado (Apache, Nginx).
  • Acceso a la terminal del servidor.
  • Permiso de administrador para instalar software.
  • Conocimientos básicos de SQL.

Instalación de MySQL en Ubuntu

  1. Abrir la terminal del servidor.
  2. Ejecutar: sudo apt update
  3. Ejecutar: sudo apt install mysql-server
  4. Iniciar el servicio: sudo systemctl start mysql

Configuración Inicial de MySQL

  • Ejecutar: sudo mysql_secure_installation
  • Seguir las indicaciones para establecer una contraseña root y configurar opciones adicionales.

Crea tu Primera Base de Datos en MySQL

  1. Ingresar a la consola de MySQL: sudo mysql -u root -p
  2. Ejecutar: CREATE DATABASE nombre_de_tu_base_de_datos;
  3. Crea un usuario y otorga permisos: GRANT ALL PRIVILEGES ON nombre_de_tu_base_de_datos.* TO 'nombre_usuario'@'localhost';

Configuración de MySQL en Apache (MyAdmin)

  • Ejecutar: sudo apt installmyadmin
  • Seguir las indicaciones para configurarMyAdmin.
  • Añadir la siguiente línea a /etc/apache2/apache2.conf: Include /etc/phpmyadmin/apache.conf

Acceso aMyAdmin

Después de la instalación, puedes acceder aMyAdmin en tu navegador: http://tu_servidor/phpmyadmin

  • Inicia sesión con el usuario y contraseña que configuraste.
  • Aquí podrás gestionar tus bases de datos y tablas.

Pruebas Básicas enMyAdmin

  1. Crea una nueva base de datos.
  2. Crea una tabla dentro de la base de datos.
  3. Añade algunos registros a la tabla.
  4. Verifica que los datos se almacenen correctamente.

Buenas Prácticas en la Configuración de SGBD

  • Usar contraseñas fuertes para el usuario root y otros usuarios.
  • Limitar los privilegios de los usuarios a lo mínimo necesario.
  • Realizar copias de seguridad periódicas de las bases de datos.

Errores Comunes en la Instalación y Configuración

  • No establecer una contraseña para el usuario root.
  • No limitar los privilegios de los usuarios.
  • No realizar copias de seguridad periódicas.

Checklist para la Instalación y Configuración

  • Instalar el SGBD (MySQL, PostgreSQL, etc.).
  • Configurar el servicio del SGBD.
  • Crea una base de datos y un usuario con privilegios adecuados.
  • Instalar y configurarMyAdmin para gestión gráfica.
  • Realizar pruebas básicas en la interfaz deMyAdmin.

Actividad: Configura tu Propio SGBD

Configura un SGBD en tu servidor web y crea una base de datos con al menos una tabla. Prueba la conexión desdeMyAdmin.
