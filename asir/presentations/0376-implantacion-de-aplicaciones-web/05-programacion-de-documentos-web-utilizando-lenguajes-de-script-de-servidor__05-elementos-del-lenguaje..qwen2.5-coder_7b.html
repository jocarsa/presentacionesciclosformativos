Introducción a los Scripts de Servidor

Los scripts de servidor son programas que se ejecutan en el lado del servidor antes de enviar una página al cliente. EJEMPLO:,, Ruby.

VARIABLES EN SCRIPTS DE SERVIDOR

Las variables en scripts de servidor se utilizan para almacenar y manipular datos. EJEMPLO: $nombre = "Juan"; echo $nombre;

Estructuras de Control

  • Condicionales (if, else)
  • Ciclos (for, while, do-while)
  • Funciones personalizadas

Inyección SQL

La inyección SQL ocurre cuando un script no valida adecuadamente la entrada del usuario, permitiendo ataques maliciosos. PRECAUCIÓN: Uso de consultas preparadas.
