Implantación de mecanismos de seguridad activa: Ataques y contramedidas en sistemas personales

Objetivo: Comprender los tipos de ataques, su análisis y las medidas preventivas y paliativas.

Clasificación de los ataques

  • Ataques de fuerza bruta
  • Inyección SQL
  • Ciberacoso
  • Phishing
  • Ransomware
  • Denegación de servicio (DoS)

Anatomía de ataques y análisis de software malicioso

Software malicioso puede ser malware, ransomware o spyware. Analizarlo requiere conocimiento eniberseguridad.

Herramientas preventivas: Instalación y configuración

  • Firewalls (pfSense, iptables)
  • Sistemas antivirus (Avast, McAfee)
  • Software de detección de intrusiones (IDS/IPS)
  • Actualización regular del software

Herramientas paliativas: Instalación y configuración

  • Software de recuperación de datos (Acronis, EaseUS)
  • Sistemas de copia de seguridad automatizados (Backblaze, Google Drive)
  • Seguridad en la nube (Dropbox, OneDrive)

Actualización de sistemas y aplicaciones

Mantener actualizados es crucial para evitar vulnerabilidades conocidas.

Seguridad en la conexión con redes públicas

  • No usar Wi-Fi público para transacciones financieras
  • Usar VPN cuando sea necesario
  • Evitar compartir información sensible

Pautas y prácticas seguras

  • Cambiar contraseñas periódicamente
  • No usar contraseñas simples o fáciles de adivinar
  • Usar autenticación de dos factores siempre que sea posible

Mini-quiz: Identificación de ataques

¿Qué tipo de ataque es el siguiente?

  1. Un sitio web que solicita tu contraseña y luego la envía a un servidor remoto.
  2. Una aplicación que se ejecuta en tu computadora sin tu conocimiento.
  3. Un correo electrónico que intenta engañarte para obtener información personal.

Checklist de seguridad personal

  • Instalar y mantener actualizado el software antivirus
  • No abrir correos electrónicos sospechosos
  • Usar contraseñas fuertes y únicas
  • Evitar compartir información personal en redes públicas
