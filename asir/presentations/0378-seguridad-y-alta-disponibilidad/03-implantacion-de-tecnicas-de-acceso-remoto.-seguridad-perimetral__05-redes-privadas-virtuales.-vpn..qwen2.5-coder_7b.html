Portada

Redes privadas virtuales (VPN)

Objetivo: Comprender los conceptos básicos y ventajas de las VPN para la seguridad perimetral.

Definición de VPN

Una VPN (Virtual Private Network) es una red privada que se establece sobre una red pública, como Internet.

Ventajas de las VPN

  • Seguridad:ifrado de datos en tránsito.
  • Acceso remoto: Permite el acceso seguro a recursos internos desde fuera.
  • Costo eficiente: Utiliza la infraestructura existente de Internet.
  • Flexibilidad: Pueden ser configuradas para diferentes tipos de accesos.

Tipo de VPNs

  • VPN de acceso remoto (Remote Access VPN): Para usuarios individuales.
  • VPN de acceso punto a punto (Site-to-Site VPN): Entre dos puntos geográficos.
  • VPN de acceso dial-up: Conexión por teléfono.

Cómo Funcionan las VPNs

Las VPNs utilizan protocolos como IPsec, SSL/TLS o PPTP para crear una capa segura sobre la red pública.

Ejemplo de Configuración

  • Cliente instala software VPN.
  • Se establece conexión con el servidor VPN.
  • Todos los tráfico entre cliente y servidor se cifra.

Caso de Uso: Trabajo Remoto

Una empresa permite a sus empleados trabajar desde casa mediante una VPN segura.

Buenas Prácticas

  • Utilizar protocolos de cifrado fuertes (IPsec, SSL/TLS).
  • Configurar políticas de acceso estrictas.
  • Mantener actualizadas las firmas del software VPN.

Errores Comunes

  • No utilizar protocolos de cifrado adecuados.
  • No configurar políticas de acceso estrictas.
  • No mantener actualizadas las firmas del software VPN.

Comparativa: VPN vs. Acceso Remoto Directo

VPN Acceso Remoto Directo
Cifrado de datos en tránsito. No garantiza cifrado.
Puede ser configurada para diferentes tipos de accesos. Acceso limitado a recursos internos.

Mini-Quiz

  1. VPN de acceso remoto
  2. VPN de acceso punto a punto
  3. VPN de acceso dial-up

Checklist Configuración VPN

  • Instalar software VPN.
  • Configurar conexión con el servidor VPN.
  • Cifrar todos los tráfico entre cliente y servidor.
  • Establecer políticas de acceso estrictas.

Actividad: Evaluación de Seguridad

Evaluación de la seguridad de una configuración VPN basada en los siguientes criterios:

  • Protocolo de cifrado utilizado.
  • Políticas de acceso estableidas.
  • Mantenimiento de actualizaciones del software.

Resumen

Las VPN son una herramienta esencial para la seguridad perimetral, proporcionando acceso remoto seguro y eficiente.

Normativa General

La normativa vigente establece requisitos mínimos de seguridad para el uso de VPNs en entornos empresariales.
