Configuración de equipos y periféricos: Arquitectura de ordenadores

Objetivo: Comprender los elementos funcionales y subsistemas de un ordenador.

Elementos Funcionales del Ordenador

  • CPU (Procesador)
  • Memoria RAM
  • Almacenamiento (HDD, SSD)
  • Pantalla o monitor
  • Teclado y ratón
  • Batería (en portátiles)

Subsistemas del Ordenador

  • Sistema Operativo
  • Controladores de dispositivos
  • Buses y puertos
  • Red y conectividad
  • Poder eléctrico y alimentación

CPU (Procesador)

El CPU es el cerebro del ordenador, ejecutando instrucciones y realizandoálculos.

Memoria RAM

RAM almacena temporalmente los datos que se están utilizando actualmente por el sistema operativo y aplicaciones.

Almacenamiento (HDD, SSD)

HDD: Disco Duro Rígido. SSD: Unidad de Estado Sólido.

Pantalla o Monitor

Muestra visualmente los datos procesados por el ordenador.

Teclado y Ratón

Permiten la entrada de información al sistema.

Batería (Portátiles)

Fuente de energía principal en dispositivos móviles.

Sistema Operativo

Software que gestiona los recursos del ordenador y proporciona una interfaz de usuario.

Controladores de Dispositivos

Software que permite a la CPU comunicarse con dispositivos periféricos.

Buses y Puertos

Canales físicos para transferir datos entre componentes del ordenador.

Red y Conectividad

Sistemas que permiten la comunicación entre dispositivos a través de redes.

Poder Eléctrico y Alimentación

Suministro de energía al ordenador para su funcionamiento.

Ejemplo: Configuración Básica de un PC

  • Instalar el sistema operativo en el HDD/SSD.
  • Configurar los controladores del CPU, RAM y dispositivos periféricos.
  • Ajustar la configuración de red para conectarse a Internet.

Caso Práctico: Diagnóstico de Problemas en el Sistema Operativo

Identificar y solucionar problemas comunes como lentitud, errores de arranque o fallos de dispositivos.

Buenas Prácticas: Mantenimiento del Ordenador

  • Limpieza regular de la superficie exterior y interna.
  • Actualizaciones periódicas del sistema operativo y software.
  • Uso de antivirus y programas de seguridad.

Errores Comunes en la Configuración

  • No instalar el sistema operativo correctamente.
  • No configurar los controladores del hardware.
  • No conectar dispositivos periféricos adecuadamente.
