Configuración de equipos y periféricos: Arquitectura de ordenadores

Objetivo: Comprender los métodos para conectar ordenadores a redes.

Definición de Redes

  • Sistema interconectado de computadoras y otros dispositivos.
  • Permite la comunicación y el interambio de datos entre diferentes equipos.
  • Tipo de conexión: inalámbrica (Wi-Fi) o por cable (Ethernet).

Ejemplos de Conexiones a Redes

  • Conexión Wi-Fi en casa.
  • Conexión Ethernet en oficina.
  • Red inalámbrica doméstica (Wi-Fi).
  • Red corporativa con múltiples dispositivos.

Tecnologías de Conexión a Redes

  • Wi-Fi 6 (802.11ax)
  • Ethernet gigabit (GigE)
  • Bluetooth
  • Z-Wave
  • LoRaWAN

Configuración Básica de Redes

Pasos para configurar una conexión Wi-Fi:

  1. Encender el ordenador.
  2. Ingresar a la pantalla de inicio.
  3. Seleccionar la opción de red.
  4. Elegir la red Wi-Fi deseada.
  5. Introducir la contraseña si es necesaria.

Configuración Avanzada de Redes

Pasos para configurar una conexión Ethernet:

  1. Conectar el cable Ethernet a la puerta del ordenador.
  2. Encender el ordenador.
  3. Ingresar a la pantalla de inicio.
  4. Seleccionar la opción de red.
  5. Elegir la conexión Ethernet.

Buenas Prácticas en Conexión a Redes

  • No compartir contraseñas de redes públicas.
  • Usar firewalls y antivirus para proteger la red.
  • Mantener actualizadas las aplicaciones del sistema operativo.
  • Evitar usar dispositivos desconocidos en la misma red.

Errores Comunes en Conexión a Redes

  • No tener conexión a Internet.
  • Problemas de velocidad de conexión.
  • Error 404 (página no encontrada).
  • Problemas con el firewall o antivirus.
  • Conexión inestable.

Mini-Quiz: Conexiones a Redes

¿Cuál de las siguientes opciones es una forma correcta de conectar un ordenador a una red Wi-Fi?

  1. Ingresar la contraseña en el navegador web.
  2. Elegir la red y introducir la contraseña si es necesario.
  3. Usar Bluetooth para conectarse.
  4. Conectar un cable Ethernet a la puerta del ordenador.

Checklist de Conexión a Redes

  • Verificar que el cable Ethernet esté conectado correctamente.
  • Asegurarse de tener la contraseña correcta para Wi-Fi.
  • Comprobar que el ordenador esté encendido.
  • Seleccionar la red correcta en las opciones disponibles.
  • Revisar si hay problemas de conexión (error 404, velocidad lenta).

Actividad: Prueba de Conexión a Redes

Realizar una prueba de conexión a red Wi-Fi en casa o en oficina.

  1. Ingresar a la pantalla de inicio del ordenador.
  2. Elegir la opción de red.
  3. Seleccionar la red Wi-Fi deseada y introducir la contraseña si es necesaria.
  4. Verificar que se tenga conexión a Internet.

Normativa Relacionada con las Redes

Reglamentos y normativas que regulan el uso de redes:

  • LGPD (Lei Geral de Proteção de Dados).
  • Normas de seguridadibernética.
  • Políticas internas de la organización sobre el uso de redes.

Caso Práctico: Seguridad en Redes

Ejemplo de caso práctico:

  • Una empresa implementa una red inalámbrica segura.
  • Se configuran firewalls y se actualizan regularmente las aplicaciones.
  • Se educan a los empleados sobre la seguridad en redes.

Caso Práctico: Problemas de Conexión a Redes

Ejemplo de caso práctico:

  • Un usuario tiene problemas con la conexión Wi-Fi.
  • Se verifica que el ordenador esté encendido y conectado a la red correcta.
  • Se reinicia el router y se verifica la velocidad de conexión.

Caso Práctico: Configuración Avanzada de Redes

Ejemplo de caso práctico:

  • Una institución educativa configura una red corporativa.
  • Se instalan firewalls y se configuran políticas de acceso.
  • Se realizan pruebas de velocidad y conexión para asegurar el rendimiento.
