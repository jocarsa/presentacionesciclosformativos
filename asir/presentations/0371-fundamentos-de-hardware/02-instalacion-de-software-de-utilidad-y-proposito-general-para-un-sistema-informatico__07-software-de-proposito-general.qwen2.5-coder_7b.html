Instalación de software de utilidad y propósito general para un sistema informático

Objetivo: Comprender los tipos de software de propósito general y su instalación en sistemas informáticos.

Software de propósito general

  • Aplicaciones diseñadas para una variedad de tareas específicas
  • No están limitadas a un solo uso específico
  • Se pueden instalar en diferentes sistemas operativos
  • Suelen ser gratuitas o con licencias económicas

Ofimática y documentación electrónica

Ejemplos: Microsoft Office, Google Docs, LibreOffice.

  • Permiten la creación, edición y gestión de documentos electrónicos
  • Son esenciales para el trabajo en entornos digitales
  • Ofrecen funciones avanzadas como formato, revisión y colaboración

Imagen, diseño y multimedia

Ejemplos: Adobe Photoshop, Canva, GIMP.

  • Permiten la creación y edición de imágenes, gráficos y videos
  • Son herramientas indispensables para el diseño web y publicidad digital
  • Ofrecen una amplia gama de filtros, efectos y herramientas de edición

Programación

Ejemplos: Visual Studio Code,Charm, IntelliJ IDEA.

  • Permiten la creación y desarrollo de software a partir del código fuente
  • Son esenciales para el desarrollo web, aplicaciones móviles y sistemas operativos
  • Ofrecen autocompletado, depuración y herramientas de control de versiones

Clientes para servicios de Internet

Ejemplos: Google Chrome, Mozilla Firefox, Microsoft Edge.

  • Permiten el acceso a navegación web y diversos servicios en línea
  • Suelen ofrecer funciones adicionales como bloqueo de publicidad y gestión de contraseñas
  • Se pueden personalizar para adaptarse a diferentes necesidades de usuario

Software a medida

Ejemplos: ERP, CRM, SIS.

  • Se desarrollan específicamente para satisfacer las necesidades de una organización
  • Suelen ser personalizables y escalables según los requisitos del cliente
  • Pueden ser adquiridos como software on-premises o en la nube

Instalación de software de ofimática

  1. Descargar el instalador desde el sitio web oficial del software
  2. Ejecutar el instalador y seguir las instrucciones
  3. Aceptar los términos y condiciones de licencia
  4. Seleccionar la ubicación de instalación
  5. Completar la configuración inicial (opcional)

Errores comunes en la instalación de software

  • No aceptar los términos y condiciones de licencia
  • Elegir una ubicación incorrecta para la instalación
  • No completar la configuración inicial
  • Instalar software incompatible con el sistema operativo

Buenas prácticas en la instalación de software

  • Aceptar siempre los términos y condiciones de licencia
  • Elegir una ubicación adecuada para la instalación
  • Completar completamente la configuración inicial
  • Instalar solo software compatible con el sistema operativo

Resumen

  • Software de propósito general es aplicativo para una variedad de tareas
  • Ejemplos incluyen ofimática, diseño gráfico, programación y clientes de Internet
  • Instalación típica implica descargar, ejecutar el instalador y seguir las instrucciones

Normativa general

La instalación de software debe ser realizada siguiendo los términos y condiciones de licencia del proveedor.

No se permite la instalación de software no autorizado o incompatible con el sistema operativo.
