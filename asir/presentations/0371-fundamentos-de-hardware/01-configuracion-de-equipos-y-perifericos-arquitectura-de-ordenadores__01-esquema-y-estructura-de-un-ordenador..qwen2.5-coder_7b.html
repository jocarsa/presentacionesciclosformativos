Esquema y estructura de un ordenador

Objetivo: Comprender la arquitectura básica de un ordenador.

Definición del ordenador

Un ordenador es una máquina electrónica diseñada para procesar información y realizar tareas automáticas.

Estructura básica de un ordenador

  • CPU (Procesador): Controla y coordina todas las operaciones del ordenador.
  • Memoria RAM: Almacena datos temporales que el CPU necesita para su procesamiento.
  • Almacenamiento permanente: Almacena programas, archivos y datos de manera persistente.
  • Pantalla (Monitor): Muestra la información visual del ordenador.
  • Teclado y ratón: Interfaz para interactuar con el usuario.

Cableado interno

Los cables internos conectan los componentes del ordenador, asegurando su funcionamiento correcto.

Ejemplo de cableado básico

  • Cable SATA para discos duros y SSDs.
  • Cable USB para dispositivos periféricos.
  • Cable PCI-E para tarjetas gráficas y expansión.
  • Cable de alimentación (ATX).

Componentes internos principales

  • Motherboard: Placa base que soporta todos los componentes.
  • Cooler: Para mantener la temperatura del procesador bajo control.
  • Batería (en portátiles): Fuente de energía alternativa.

Ejemplo de configuración de un ordenador básico

  • CPU Intel Core i5
  • 8 GB RAM DDR4
  • 1 TB HDD
  • 1600x900 Monitor LCD
  • Teclado y ratón inalámbricos

Proceso de instalación del sistema operativo

  1. Preparar el medio de instalación (USB o DVD).
  2. Iniciar el ordenador desde el medio de instalación.
  3. Elegir idioma y configuraciones iniciales.
  4. Instalar el sistema operativo.

Buenas prácticas en la configuración del ordenador

  • Mantener los cables organizados y libres de entrelazamientos.
  • Evitar sobrecalentamiento al colocar el ordenador en un lugar fresco y ventilado.
  • Usar protectora para evitar daños por caídas o golpes.

Errores comunes en la configuración del ordenador

  • No conectar el cable de alimentación correctamente.
  • Montar componentes en posiciones incorrectas.
  • No instalar drivers o software necesarios.

Checklist de configuración del ordenador

  1. Verificar la conexión de todos los cables.
  2. Asegurarse de que el cableado esté organizado.
  3. Comprobar que todos los componentes están correctamente instalados.
  4. Revisar la temperatura del procesador.

Actividad: Diseño de un ordenador básico

Diseña un ordenador básico con los siguientes requisitos:

  • CPU Intel Core i3
  • 4 GB RAM DDR4
  • 500 GB HDD
  • 1920x1080 Monitor LCD
  • Teclado y ratón inalámbricos

Resumen de la arquitectura del ordenador

El ordenador se compone de componentes interconectados que trabajan juntos para procesar información y realizar tareas.

Normativa en la configuración del ordenador

  • Seguridad eléctrica: Protección contra cortocircuitos y sobrecalentamiento.
  • Compatibilidad: Asegurar que todos los componentes sean compatibles entre sí.
  • Mantenimiento regular: Limpieza y actualizaciones de software.
