Portada

Realización de consultas en bases de datos

Objetivo: Aprender a seleccionar y ordenar registros, así como manejar valores nulos.

Selección de Registros

  • Utiliza la cláusula SELECT para especificar los campos que deseas recuperar.
  • Ejemplo: SELECT nombre, edad FROM usuarios;
  • Podrías seleccionar todos los registros con *: SELECT * FROM usuarios;

Filtros con WHERE

La cláusula WHERE se utiliza para filtrar registros según una condición.

  • Ejemplo: SELECT nombre FROM usuarios WHERE edad > 18;
  • Puedes combinar múltiples condiciones con AND, OR y NOT.

Ordenación de Registros

La cláusula ORDER BY se utiliza para ordenar los resultados de una consulta.

  • Ejemplo: SELECT nombre FROM usuarios ORDER BY edad ASC;
  • Puedes ordenar en orden ascendente (ASC) o descendente (DESC).

Manejo de Valores Nulos

Los valores nulos representan ausencia de datos.

  • Para verificar si un valor es nulo, usa IS NULL o IS NOT NULL.
  • Ejemplo: SELECT nombre FROM usuarios WHERE edad IS NULL;

Funciones Agregadas

Las funciones agregadas realizanálculos sobre un conjunto de valores.

  • Ejemplo: SELECT AVG(edad) FROM usuarios;
  • Otras funciones: COUNT, SUM, MAX, MIN.

Aliases

Puedes asignar nombres temporales a los resultados de una consulta.

  • Ejemplo: SELECT nombre AS NombreUsuario FROM usuarios;

Subconsultas

Una subconsulta es una consulta dentro de otra consulta.

  • Ejemplo: SELECT nombre FROM usuarios WHERE id IN (SELECT usuario_id FROM pedidos);

Buenas Prácticas

  • Evita usar * en consultas grandes para mejorar el rendimiento.
  • Utiliza índices en columnas que se filtran frecuentemente.
  • Optimiza las consultas utilizando JOINs eficientes.

Errores Comunes

  • Sintaxis incorrecta: Verifica los comandos SQL.
  • No se especifican campos en SELECT: Recuerda incluir todos los campos necesarios.
  • Olvidar ORDER BY: Los resultados pueden no estar ordenados según lo esperado.

Caso Práctico

Ejemplo de consulta compleja:

SELECT u.nombre, AVG(p.importe) AS promedio_compra
FROM usuarios u
JOIN pedidos p ON u.id = p.usuario_id
WHERE p.fecha > '2023-01-01'
GROUP BY u.id
ORDER BY promedio_compra DESC;

Mini-Quiz

¿Cuál de las siguientes consultas es correcta?

  • A) SELECT nombre FROM usuarios WHERE edad > 18 ORDER BY nombre DESC;
  • B) SELECT nombre, edad FROM usuarios WHERE edad IS NULL;
  • C) SELECT * FROM usuarios WHERE edad = '25';

Checklist

  • Verifica la sintaxis de las consultas.
  • Asegúrate de incluir los campos necesarios en SELECT.
  • Ordena los resultados según lo necesario con ORDER BY.
  • Maneja valores nulos correctamente con IS NULL o IS NOT NULL.

Actividad

Escribe una consulta que seleccione el nombre y la edad de los usuarios mayores de 30 años, ordenados por edad en orden descendente.

Resumen

  • Selección de registros con SELECT.
  • Filtros con WHERE.
  • Ordenación con ORDER BY.
  • Manejo de valores nulos.

Normativa General

No hay normativas específicas, pero sigue buenas prácticas y asegúrate de que tus consultas sean eficientes y seguras.
