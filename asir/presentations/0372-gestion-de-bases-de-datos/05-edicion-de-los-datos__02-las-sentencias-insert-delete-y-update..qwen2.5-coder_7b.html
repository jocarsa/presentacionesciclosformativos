Las Sentencias INSERT, DELETE y UPDATE

  • INSERT: Añade nuevos registros a una tabla.
  • DELETE: Elimina registros existentes de una tabla.
  • UPDATE: Modifica los valores de los registros en una tabla.

Ejemplo de INSERT

INSERT INTO empleados (nombre, apellido, edad) VALUES ('Juan', 'Pérez', 30);

Ejemplo de DELETE

DELETE FROM empleados WHERE id = 1;

Ejemplo de UPDATE

UPDATE empleados SET edad = 31 WHERE id = 1;

Buenas Prácticas

  • Usar claves primarias para identificar registros.
  • Validar los datos antes de insertar o actualizar.
  • Realizar transacciones en bloques para mantener la integridad.

Errores Comunes

  • No especificar las columnas a insertar (INSERT INTO empleados VALUES ('Juan', 'Pérez');)
  • Eliminar todos los registros sin condición (DELETE FROM empleados;)
  • No comprobar la existencia de datos antes de actualizar.

Normativa General

No se permite la eliminación masiva de datos sin condición.

Se debe realizar una copia de seguridad antes de realizar cambios significativos en los datos.
