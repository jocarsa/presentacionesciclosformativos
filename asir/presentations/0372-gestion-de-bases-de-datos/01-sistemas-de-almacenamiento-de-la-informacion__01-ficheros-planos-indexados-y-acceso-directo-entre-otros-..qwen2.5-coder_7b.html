Concepto de Fichero

Un fichero es una colección organizada de datos almacenados en un dispositivo de almacenamiento.

Tipo de Ficheros: Planos

  • No estructurados
  • Datos almacenados secuencialmente
  • Acceso secuencial
  • Ejemplo: Archivos de texto plano (.txt)

Tipo de Ficheros: Indexados

  • Estructurados
  • Datos organizados en tablas
  • Acceso directo a registros
  • Ejemplo: Archivos CSV (.csv)

Tipo de Ficheros: Acceso Directo

  • Estructurados
  • Datos organizados en tablas
  • Acceso directo a registros mediante índices
  • Ejemplo: Archivos de base de datos (.db)

Ventajas del Acceso Directo

  • Velocidad en el acceso a registros individuales
  • Eficiencia en la actualización y eliminación de datos
  • Facilidad en la organización y consulta de datos complejos

Inventarios de Bases de Datos

Un inventario de bases de datos es una lista detallada de todas las bases de datos existentes en un sistema.

Ejemplo de Inventario de Bases de Datos

  • Nombre de la base de datos
  • Tipo de base de datos (SQL, NoSQL)
  • Número de registros
  • Última fecha de actualización

Caso Práctico: Gestión de Inventario

Una empresa puede utilizar una base de datos para gestionar su inventario, almacenando información sobre productos, proveedores y ventas.

Buenas Prácticas en la Gestión de Bases de Datos

  • Regular respaldos periódicos
  • Optimizar consultas frecuentes
  • Mantener índices actualizados
  • Monitorear el rendimiento del sistema

Errores Comunes en la Gestión de Bases de Datos

  • No respaldar regularmente las bases de datos
  • No optimizar consultas
  • No mantener índices actualizados
  • No monitorear el rendimiento del sistema
