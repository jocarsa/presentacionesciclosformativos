Realización de consultas en bases de datos

Objetivo: Aprender a utilizar la sentencia SELECT para consultar datos en una base de datos.

¿Qué es la sentencia SELECT?

La sentencia SELECT se utiliza para seleccionar datos de una o varias tablas en una base de datos.

Sintaxis básica de SELECT

SELECT column1, column2, ...
FROM table_name;

Ejemplo básico

Seleccionar todos los registros de la tabla "Empleados":

SELECT * FROM Empleados;

Seleccionar columnas específicas

Ejemplo: Seleccionar solo el nombre y el salario de los empleados:

SELECT Nombre, Salario FROM Empleados;

Filtros con WHERE

Añadir un filtro para seleccionar solo registros que cumplan ciertas condiciones.

SELECT * FROM Empleados WHERE Departamento = 'Ventas';

Ordenamiento con ORDER BY

Ordenar los resultados de una consulta.

SELECT Nombre, Salario FROM Empleados ORDER BY Salario DESC;

Límites con LIMIT

Limitar el número de registros devueltos por una consulta.

SELECT Nombre, Salario FROM Empleados ORDER BY Salario DESC LIMIT 10;

Funciones agregadas

  • SUM(columna): Suma los valores de una columna.
  • AVG(columna): Calcula el promedio de los valores de una columna.
  • COUNT(columna):uenta el número de registros que cumplen la condición.

Mini-quiz

¿Cuál es la diferencia entre SELECT * y SELECT Nombre, Salario?

  1. Devuelven todos los registros de la tabla.
  2. Devuelven solo el nombre y el salario de los empleados.
  3. No hay diferencia, son equivalentes.

Comprobación del mini-quiz

La respuesta correcta es: Devuelven solo el nombre y el salario de los empleados.

Buenas prácticas

  • Usar nombres descriptivos para las columnas.
  • Añadir índices a las columnas utilizadas en WHERE, ORDER BY y JOIN.
  • Evitar SELECT * cuando solo se necesiten algunas columnas.

Errores comunes

  • Olvidar la cláusula FROM.
  • Escribir mal el nombre de una columna o tabla.
  • No usar paréntesis alrededor de las funciones agregadas.

Comparativa con otras sentencias SQL

Sentencia Descripción
INSERT INTO Inserta nuevos registros en una tabla.
UPDATE Actualiza los valores de las columnas de un registro o registros.
DELETE FROM Borra registros de una tabla.

Checklist

  • ¿Se ha especificado correctamente la tabla?
  • ¿Se han seleccionado las columnas correctas?
  • ¿Se ha aplicado el filtro necesario?
  • ¿Se ha ordenado los resultados de manera adecuada?
  • ¿Se ha limitado el número de registros devueltos?

Ejercicios prácticos

Realizar consultas SQL para seleccionar datos de una base de datos ficticia.

  1. Selecciona todos los clientes de la ciudad de Madrid.
  2. Calcula el promedio del salario de los empleados en el departamento de Finanzas.
  3. Ordena los productos por precio descendente y limita el resultado a 5.

Resumen

  • La sentencia SELECT se utiliza para consultar datos en una base de datos.
  • Se pueden seleccionar todas las columnas o solo algunas específicas.
  • Pueden aplicarse filtros, ordenamientos y límites a las consultas.

Normativa general

No hay normativas específicas para la sentencia SELECT en SQL. Sin embargo, es importante seguir buenas prácticas de codificación y seguridad al realizar consultas.
