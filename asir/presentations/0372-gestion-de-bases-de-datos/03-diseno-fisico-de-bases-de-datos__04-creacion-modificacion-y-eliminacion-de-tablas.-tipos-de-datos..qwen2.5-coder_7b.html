Tipos de Datos en Bases de Datos

  • Numericos: INT, FLOAT, DOUBLE
  • Cadenas de texto: VARCHAR, CHAR
  • Fecha y hora: DATE, TIME, DATETIME
  • Lógicos: BOOLEAN
  • Enumerados: ENUM

Ejemplo de Creación de Tabla

CREATE TABLE empleados (
    id INT PRIMARY KEY,
    nombre VARCHAR(100),
    apellido VARCHAR(100),
    fecha_nacimiento DATE,
    salario FLOAT
);

  • Añadir columnas: ALTER TABLE ... ADD COLUMN

Ejemplo de Modificación de Tabla

Buenas Prácticas al Crear Tablas

  • Nombre descriptivo para las tablas y columnas

Errores Comunes al Crear Tablas

  • No definir claves primarias
  • Usar tipos de datos incorrectos
  • No considerar restricciones de integridad
  • Omitir índices importantes
